# コメントを読むな

- 原文URL: https://blog.samaltman.com/dont-read-the-comments
- 原題: Don't Read The Comments

> 注: 自動翻訳をベースに、読みやすいよう最小限の整形を行っています。

今朝、現在のYCバッチにこのメールを送った：

ネガティブなメディア報道や自社に対するオンラインコメントについて、本当に落ち込んでいる人たちと話した。たいていの場合、「ああ、最近の新しいスタートアップはみんなダメだ、良いものはもう全部始まっている」という趣旨のものだ。

アンチがいるのは辛いことだが、今や巨大な会社を経営している創業者は全員、長い間これに直面してきた。どうか落ち込まないでほしい（批判の中には有益なものもあり、それには注意を払うべきだが、普段人を落ち込ませるのはそういう種類のものではない）。これに対する厚い皮を早く身につけられれば、それだけ良い。

世界がすぐに終わらない限り、世界がこれまでに見る中で最も価値のある企業はまだ始まっていない。

ほとんどのスタートアップは失敗するので、「全部ダメだ」と言えばほとんどの場合正しくなれる。その戦略で損をすることはないが、利益を得ることもない。

最高のスタートアップが良いと認められるまでには長い時間がかかる。Google、Facebook、Airbnb、Uberなどについて、創業から最初の数年間に人々が書いたことを読んでみるといい。一夜の成功には通常、10年間の上り坂の努力が必要だ。

YC自体もこれに長い間直面してきた。結果的にはうまくいった。

私の友人がよく言う言葉がある。「世界には2種類の人間がいる——未来を作る人間と、なぜ彼らが失敗するかについてインターネットに投稿を書く人間だ。」前者のカテゴリーに入るよう努力し続けてほしい。

世界にはもう新しいことはないと言った人たちは、毎回間違っていた。彼らの想像力の欠如に自分を引き止めさせないでほしい。
