# テックワーカーの価値観

良くも悪くも、テクノロジーは私たち全員の生活における中心的な力となった。

コミュニティのメンバーとして、テクノロジー企業がその集合的な力を使って、プライバシー、法の支配、表現の自由、その他の基本的なアメリカの権利を守る方法に関心がある。

また、テクノロジー企業がいかにして国の分断を癒せるかについても議論したい。テクノロジー企業は、高収入のテクノロジー関連の仕事を全国に広げることやその他の施策を通じて、すべてのアメリカ人にとってより良い経済的未来を創り出せると信じている。また、テクノロジー企業には、自分たちが助長した分極化を減らす機会と義務があると信じている。

テクノロジー企業は従業員の影響力に非常に敏感だ。従業員は団結し、自分たちの企業に守ってほしい価値観と方針を明確に定義すべきだと考えている。テック労組という比喩は完全には当てはまらないが、それほど的外れでもない。
