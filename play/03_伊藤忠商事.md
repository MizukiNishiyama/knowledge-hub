## 2. 組織構造・階層

### ディビジョンカンパニー制（1997年導入）

伊藤忠商事は **8つのディビジョンカンパニー＋総本社** で構成。各カンパニーは独立採算制を持つ「社内カンパニー」。

```
会長CEO / 社長COO
├── 繊維カンパニー（ファッションアパレル、ブランドマーケティング）
├── 機械カンパニー（プラント・船舶・航空機・自動車）
├── 金属カンパニー（金属資源、鉄鋼製品）
├── エネルギー・化学品カンパニー（エネルギートレード、化学品）
├── 食料カンパニー（食糧、生鮮食品、食品流通）
├── 住生活カンパニー（建材・住宅、パルプ・紙）
├── 情報・金融カンパニー（IT・通信、保険・金融）
├── 第8カンパニー（2019年7月設立、消費者向け新ビジネス）
└── 総本社（全社戦略の策定・総合管理）
    └── コーポレート
        ├── 経営企画部
        ├── 人事・総務部
        ├── 経理部
        ├── 法務部
        ├── IT企画部
        └── 広報部
```

### 職位階層

```
会長CEO → 社長COO → 副社長 → 専務執行役員
→ 常務執行役員 → 執行役員
→ カンパニープレジデント → 部門長 → 部長
→ 課長 → 課長代行 → 担当
```

### 特徴的な組織文化

- **朝型勤務制度**: 20時以降の残業を原則禁止、早朝出勤を奨励
- **厳しい成果主義**: 商社の中で最もシビアな評価制度
- **「稼ぐ、削る、防ぐ」**: 岡藤会長CEOが掲げた経営哲学

## 5. 新卒入社後のキャリアパス

### グレード制度（Band/Grade 1〜7）

| グレード | 想定役職 | 年収レンジ |
|---------|---------|-----------|
| Band 1〜3 | 担当者（スタッフ） | 〜最高2,500万円 |
| Band 4 | 課長代行 | 最高2,970万円 |
| Band 5 | 課長 | 最高**3,620万円** |
| Band 6 | 部長 | 最高**4,110万円** |
| Band 7 | 本部長・カンパニープレジデント | それ以上 |

※全社平均年収: **1,805万円**（2025年3月期）

### 入社〜3年目（基礎教育期間）

```
入社 → 「原籍組織」が決定（原則として原籍は変わらない）
  → 入社直後〜1年目: 多くの場合、子会社・グループ会社へ出向
  → 子会社で「モノの流れ」「お金の流れ」を現場で学ぶ
  → 「基礎教育は2年まで」のガイドライン
```

- **新卒採用数**: 毎年約100〜130名（総合職）
- **朝型勤務**: 8時出勤が基本。20時退社を徹底

### 3年目〜8年目（海外駐在・専門性形成）

- **「原則8年以内での海外派遣」** を前提としたローテーションガイドライン
- 海外駐在先は現地法人や投資先子会社
- 若手は現場のプレーヤー（Sales等）として赴任
- 全社員の約**20%**が海外駐在員

### 中国語人材の育成

- CITIC/CP提携に伴い、全総合職の1/3にあたる**「1,000人の中国語人材」**育成プロジェクト
- 2024年度末時点で通算育成数は**1,330人**に到達

### 中堅以降（課長・部長へ）

- 中堅は海外現地法人でマネジャー（Sales Manager等）として赴任
- ベテランは経営層（Managing Director、Director等）として赴任
- 子会社出向で得た現場経験を「投資業務」「マーケティング」に活かす
- 昇進は実力次第で30代前半の課長もあり（他商社より昇進が早い傾向）

## 6. キャリアパス階層（上位X%ごとの到達点）

### 昇進ピラミッド

```
                    ┌──────────┐
                    │  社長・会長 │  ← 同期の約0.5%未満
                    └────┬─────┘
                   ┌─────┴──────┐
                   │  副社長・専務 │  ← 上位約1〜2%
                   └─────┬──────┘
                  ┌──────┴───────┐
                  │   執行役員    │  ← 上位約3〜5%
                  └──────┬───────┘
              ┌──────────┴──────────┐
              │   部長(Band6)      │  ← 上位約5〜10%
              └──────────┬──────────┘
          ┌──────────────┴──────────────┐
          │    課長(Band5)             │  ← 上位約15〜25%
          └──────────────┬──────────────┘
      ┌──────────────────┴──────────────────┐
      │    課長代行以下(Band1〜4)         │  ← 残り
      └────────────────────────────────────┘
```

### 到達点の目安

| レベル | 割合（概算） | 典型的なキャリア |
|--------|-------------|----------------|
| **トップ0.5%** | 約1名/年 | 社長候補。カンパニープレジデント複数歴任 |
| **上位1〜3%** | 2〜4名/年 | 執行役員。カンパニープレジデントや海外総支配人 |
| **上位5〜10%** | 5〜10名/年 | 部長（Band 6）。年収最高4,110万円 |
| **上位10〜20%** | 10〜20名/年 | 子会社社長・取締役。ファミマやCTC等の経営層 |
| **上位20〜40%** | 20〜40名/年 | 課長（Band 5）。年収最高3,620万円 |
| **上位40〜100%** | — | 関連会社、転職、独立 |

### 子会社社長ポストの特徴

- 連結対象**263社**: 子会社社長ポストは豊富
- ファミリーマート: 社長は伊藤忠からの派遣（執行役員級）
- CTC: 社長は伊藤忠出身者が多い
- **伊藤忠の特徴**: 成果主義が徹底しており、出世が早い人と遅い人の差が大きい

### 成果主義の徹底

> 「働いても働かんでも、そこそこもらえたら誰も仕事せん。もっとお金に差をつけて闘争本能を刺激せなあかん」（岡藤会長CEO）

- 2025年度からの報酬制度改定で、同じ役職でも成績上位と下位で年収に大きな差がつく仕組みを強化
- 2024年度は年収1億円超の役員が複数（五大商社の中でも上位を独占）
