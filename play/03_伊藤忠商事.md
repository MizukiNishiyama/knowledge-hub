# 伊藤忠商事（ITOCHU Corporation）

> 「非資源No.1」の商社。生活消費関連ビジネスに圧倒的な強みを持ち、近年は時価総額で三菱商事と首位を争う。岡藤正広会長のリーダーシップのもと「稼ぐ、削る、防ぐ」を徹底。

## 2. 組織構造・階層

### ディビジョンカンパニー制

伊藤忠商事は **8つのディビジョンカンパニー** を中核とした組織。

```
会長CEO / 社長COO
├── 繊維カンパニー
├── 機械カンパニー
├── 金属カンパニー
├── エネルギー・化学品カンパニー
├── 食料カンパニー
├── 住生活カンパニー
├── 情報・金融カンパニー
├── 第8カンパニー
└── コーポレート
    ├── 経営企画部
    ├── 人事・総務部
    ├── 経理部
    ├── 法務部
    ├── IT企画部
    └── 広報部
```

### 職位階層

```
会長CEO → 社長COO → 副社長 → 専務執行役員
→ 常務執行役員 → 執行役員
→ カンパニープレジデント → 部門長 → 部長
→ 課長 → 課長代理 → 担当
```

### 特徴的な組織文化

- **朝型勤務制度**: 20時以降の残業を原則禁止、早朝出勤を奨励
- **厳しい成果主義**: 商社の中で最もシビアな評価制度と言われる
- **「稼ぐ、削る、防ぐ」**: 岡藤会長が掲げた経営哲学

## 5. 新卒入社後のキャリアパス

### 入社〜5年目（若手期）

```
入社 → 配属先決定（8カンパニーのいずれか or コーポレート）
  → 1〜2年目: OJT、実務を通じた即戦力育成
  → 3年目前後: 海外トレーニー or 国内子会社研修
  → 4〜5年目: 主担当。成果を出すことが強く求められる
```

- **新卒採用数**: 毎年約130〜150名（総合職）
- **朝型勤務**: 8時出勤が基本。20時退社を徹底
- **成果主義**: 若手でも成果次第で重要案件を任される

### 6年目〜15年目（中堅期）

- **海外駐在**: 約50〜60%が経験
  - 主要駐在先: 中国、東南アジア、北米、欧州
  - 期間: 3〜5年
- **子会社出向**: ファミリーマート、CTC、日本アクセスなど
  - 実際の事業経営を現場で学ぶ
- **昇進**: 課長（30代後半〜）。実力次第では30代前半も

### 15年目以降（管理職期）

- **部長**: 40代前半〜（他商社より昇進が早い傾向）
- **カンパニー内異動**: 専門性を深めるケースが多い
- **執行役員**: 40代後半〜50代前半

## 6. キャリアパス階層（上位X%ごとの到達点）

### 昇進ピラミッド

```
                    ┌──────────┐
                    │  社長・会長 │  ← 同期の約0.5%未満
                    └────┬─────┘
                   ┌─────┴──────┐
                   │  副社長・専務 │  ← 上位約1〜2%
                   └─────┬──────┘
                  ┌──────┴───────┐
                  │   執行役員    │  ← 上位約3〜5%
                  └──────┬───────┘
              ┌──────────┴──────────┐
              │   部長・部門長級    │  ← 上位約10〜15%
              └──────────┬──────────┘
          ┌──────────────┴──────────────┐
          │        課長級              │  ← 上位約30〜40%
          └──────────────┬──────────────┘
      ┌──────────────────┴──────────────────┐
      │      課長代理・担当              │  ← 残り
      └────────────────────────────────────┘
```

### 到達点の目安

| レベル | 割合（概算） | 典型的なキャリア |
|--------|-------------|----------------|
| **トップ0.5%** | 約1名/年 | 社長候補。カンパニープレジデント複数歴任 |
| **上位1〜3%** | 3〜5名/年 | 執行役員。カンパニープレジデントや海外総支配人 |
| **上位5〜10%** | 7〜13名/年 | 部長・部門長。カンパニー内の中核 |
| **上位10〜20%** | 13〜25名/年 | 子会社社長・取締役。ファミマやCTC等の経営層 |
| **上位20〜40%** | 25〜50名/年 | 課長として専門分野で活躍 |
| **上位40〜100%** | — | 関連会社、転職、独立 |

### 子会社社長ポストの特徴

- ファミリーマート: 社長は伊藤忠からの派遣（執行役員級）
- CTC: 社長は伊藤忠出身者が多い
- 連結子会社約300社: 子会社社長ポストは豊富
- **伊藤忠の特徴**: 成果主義が徹底しており、出世が早い人と遅い人の差が大きい

## 7. 伊藤忠商事の強み・特徴

### 非資源ビジネスの王者
- 資源価格に左右されにくい安定的な収益構造
- ファミリーマート（約16,500店舗）を核とした生活消費ビジネス
- 川上（原料）から川下（小売）まで一貫したバリューチェーン

### 「商人の伊藤忠」
- 非財閥系ゆえの機動力と柔軟性
- 関西商人のDNA: 利益へのこだわりが強い
- 岡藤正広会長の強力なリーダーシップ

### 中国ビジネスの強さ
- CITIC（中国中信集団）への約6,000億円の大型投資（2015年）
- 中国最大の国有コングロマリットとの戦略的パートナーシップ
- 中国市場へのアクセスで他商社を圧倒

### 高い資本効率
- ROE: 五大商社の中でトップクラス（15%超）
- 株主還元: 積極的な自社株買い・増配
- PBR: 商社の中で最も高い評価
