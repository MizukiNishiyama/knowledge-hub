# 丸紅（Marubeni Corporation）

> 「電力の丸紅」「穀物の丸紅」。海外IPP（独立系発電事業）と穀物トレーディングで世界トップクラス。伊藤忠商事と同根（伊藤忠兵衛商店から分離）。チャレンジングな社風。

---

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| 設立 | 1949年（前身は1858年の伊藤忠兵衛商店。1921年に丸紅商店として分離） |
| 本社 | 東京都千代田区大手町 |
| 従業員数 | 単体 約4,400名 / 連結 約50,000名 |
| 時価総額 | 約4〜5兆円（2025年3月期時点） |
| 特徴 | 非財閥系、伊藤忠と同根、電力・穀物が強い |

## 2. 組織構造・階層

### グループ制（事業グループ制）

丸紅は **13の営業グループ** を中心に組織を構成。

```
社長・CEO
├── ライフスタイルグループ
├── 情報・不動産グループ
├── フォレストプロダクツグループ
├── 食料グループ（第一・第二）
├── アグリ事業グループ
├── 化学品グループ
├── 電力グループ（第一・第二）
├── エネルギーグループ
├── 金属グループ
├── プラントグループ
├── 航空・船舶グループ
├── 金融・リース・不動産グループ
├── CDIO（次世代事業開発）
└── コーポレートスタッフグループ
    ├── 経営企画部
    ├── 人事部
    ├── 財務部
    ├── 法務部
    ├── リスクマネジメント部
    └── サステナビリティ推進部
```

### 職位階層

```
会長 → 社長 → 副社長 → 専務執行役員
→ 常務執行役員 → 執行役員
→ グループCEO → 本部長 → 部長
→ 課長 → 担当
```

## 5. 新卒入社後のキャリアパス

### 入社〜5年目（若手期）

```
入社 → 配属先決定（13グループ or コーポレート）
  → 1〜2年目: OJT中心。トレーディング業務
  → 3年目前後: 海外研修制度
  → 4〜5年目: 担当案件の主力。海外出張も増加
```

- **新卒採用数**: 毎年約100〜120名（総合職）
- **社風**: 体育会系の雰囲気が残るが、チャレンジングで自由
- **少数精鋭**: 五大商社の中で単体従業員が最も少ない部類

### 6年目〜15年目（中堅期）

- **海外駐在**: 約50〜60%が経験
  - 主要駐在先: 東南アジア、北米、中南米、アフリカ
  - 電力事業は特に中南米・アフリカなど新興国駐在が多い
  - 期間: 3〜5年
- **子会社出向**: 各電力事業会社、穀物事業会社
- **昇進**: 課長（30代後半〜40代前半）

### 15年目以降（管理職期）

- **部長**: 40代半ば〜
- **グループCEO**: 50代前半〜
- **執行役員**: 50代半ば〜

## 6. キャリアパス階層（上位X%ごとの到達点）

### 昇進ピラミッド

```
                    ┌──────────┐
                    │  社長・会長 │  ← 同期の約0.5%未満
                    └────┬─────┘
                   ┌─────┴──────┐
                   │  副社長・専務 │  ← 上位約1〜2%
                   └─────┬──────┘
                  ┌──────┴───────┐
                  │   執行役員    │  ← 上位約3〜5%
                  └──────┬───────┘
              ┌──────────┴──────────┐
              │   部長・本部長級    │  ← 上位約10〜15%
              └──────────┬──────────┘
          ┌──────────────┴──────────────┐
          │        課長級              │  ← 上位約35〜45%
          └──────────────┬──────────────┘
      ┌──────────────────┴──────────────────┐
      │          担当                     │  ← 残り
      └────────────────────────────────────┘
```

### 到達点の目安

| レベル | 割合（概算） | 典型的なキャリア |
|--------|-------------|----------------|
| **トップ0.5%** | 約1名/年 | 社長候補。複数グループCEOを歴任 |
| **上位1〜3%** | 2〜4名/年 | 執行役員。グループCEOや海外統括 |
| **上位5〜10%** | 5〜10名/年 | 部長級。事業の責任者 |
| **上位10〜20%** | 10〜20名/年 | 子会社社長・取締役 |
| **上位20〜45%** | 20〜50名/年 | 課長。実務のスペシャリスト |
| **上位45〜100%** | — | 関連会社・転職 |

### 子会社社長ポスト

- 連結子会社約280社
- 電力事業会社の社長ポストが特に多い（20カ国以上）
- 穀物事業会社（Gavilon後継）の幹部ポスト
- 少数精鋭のため、一人当たりのチャンスは比較的多い

## 7. 丸紅の強み・特徴

### 「電力の丸紅」
- 海外IPP事業で世界トップクラスの実績
- 持分発電容量約12GW
- 再エネへの転換を積極推進
- 新興国での電力インフラ整備に強み

### 「穀物の丸紅」
- 2013年のGavilon買収（約2,800億円）で穀物事業を大幅強化
- 世界の穀物バリューチェーンに深く関与
- 食料安全保障の観点からも重要な事業

### チャレンジングな企業文化
- 五大商社の中で最も「攻め」の姿勢
- 新規事業への果敢な投資
- 2020年に約2,000億円の大規模減損（資源・穀物）を経験し、体質を改善

### 「丸紅新プラン」による変革
- 2024年に「GC2027」（中期経営計画）を発表
- グリーン戦略とDXを柱に据える
- 電力事業の再エネシフトを加速
