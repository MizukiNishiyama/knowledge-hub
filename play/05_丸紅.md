# 丸紅（Marubeni Corporation）

> 「電力の丸紅」「穀物の丸紅」。海外IPP（独立系発電事業）と穀物トレーディングで世界トップクラス。伊藤忠商事と同根（伊藤忠兵衛商店から分離）。2025年4月に16本部制から10部門制へ大規模組織改革。

---

---

## 2. 組織構造・階層

### 10部門制（2025年4月〜、16本部制から大規模再編）

丸紅は **2025年4月1日付** で従来の16営業本部体制を **10営業部門** に再編。同時に「グループCEO」を廃止し、新たに「**統括役員**」を設置。

```
社長・CEO（大本晶之）
├── 1. ライフスタイル部門（旧ライフスタイル本部＋フォレストプロダクツ本部）
├── 2. 食料・アグリ部門（旧食料第一＋第二＋アグリ事業本部）
├── 3. 金属部門（旧金属本部）
├── 4. エネルギー・化学品部門（旧エネルギー＋化学品本部）
├── 5. 電力・インフラサービス部門（旧電力＋インフラプロジェクト＋新エネルギー開発推進部）
├── 6. 金融・リース・不動産部門（旧金融・リース・不動産本部）
├── 7. エアロスペース・モビリティ部門（旧航空・船舶＋建機・産機・モビリティ本部）
├── 8. 情報ソリューション部門（旧情報ソリューション本部）
├── 9. 次世代事業開発部門（旧次世代事業開発本部）
├── 10. 次世代コーポレートディベロップメント部門
├── バリュークリエーションオフィス（VCO）← 社長直下の新設組織
└── コーポレート
    ├── 経営企画部
    ├── 人事部
    ├── 財務部
    ├── 経理部
    ├── 法務部
    ├── リスクマネジメント部
    └── サステナビリティ推進部
```

### 組織改革のポイント
- 各営業部門に「**戦略企画部**」を新設
- 全営業部門（次世代CD除く）に「**成長投資マネジメント室**」を新設
- 2026年6月に **指名委員会等設置会社** への移行を計画

### 職位階層・グレード制度

```
会長 → 社長 → 副社長 → 専務執行役員
→ 常務執行役員 → 執行役員（Tグレード）
→ 部長（Sグレード） → 課長・部長代理（M4〜M5）
→ 主任・係長・課長代理（M1〜M3） → 担当（J1〜J8）
```

## 5. 新卒入社後のキャリアパス

### グレード制度と年収レンジ

| グレード | 役職相当 | 年次目安 | 年収レンジ |
|---------|---------|---------|-----------|
| J1〜J8 | 一般職（担当〜主任） | 新卒1〜8年目 | 500〜1,200万円 |
| M1〜M3 | 主任・係長・課長代理 | 9〜15年目 | 1,400〜2,000万円 |
| M4〜M5 | 課長・部長代理 | 15〜20年目 | 1,800〜2,500万円 |
| Sグレード | 部長 | 20年目以降 | 2,500〜3,000万円 |
| Tグレード | 本部長・執行役員 | — | 3,000万円超 |

※Jグレードでは毎年約100万円昇給、**30歳前後で年収1,000万円超**が一般的
※海外駐在があれば**30代後半で年収2,000万円**も可能

### 入社〜5年目（若手期・J1〜J5）

```
入社 → 配属先決定（10部門 or コーポレート）
  → 1〜3年目: 本社配属 or 子会社出向。基礎業務を学ぶ
  → 3〜5年目: 海外語学研修・実務研修
```

- **新卒採用数**: 毎年約100〜120名（総合職）
- **Career Vision採用**: 2021年卒から導入。配属部署・業務を明示した募集
- **少数精鋭**: 五大商社の中で単体従業員が最も少ない部類
- **20代で関連会社の取締役に就任する例もある**

### 5年目〜12年目（中堅期・J5〜M3）

- **海外駐在（1回目）**: 3〜5年間。若手のうちから海外拠点で実務
  - 電力事業は特に中南米・アフリカなど新興国駐在が多い
- **M グレード昇格**: 管理職に。プロジェクトリーダーを担当

### 12年目〜20年目（管理職期・M3〜M5）

- **海外駐在（2回目）or 子会社出向**: より大きな裁量を持つポジション
- 子会社取締役に就任するケースも
- **課長・部長代理（M4〜M5）**: 投資案件の意思決定や部下のマネジメント

### 20年目以降（部長〜・Sグレード以上）

- **部長（Sグレード）**: 部門の中核
- **執行役員（Tグレード）**: 事業の最終責任者

### 特徴的な制度
- **丸紅キャリア・ディベロップメント・プログラム**: 毎年、社員本人が希望キャリアプランを申告
- **公募異動制度**: 社内ポストに自ら手を挙げて異動可能
- **MBA留学制度**: 海外ビジネススクールへの派遣
- **社内起業・ベンチャー出向**: 新規事業創出の機会あり

## 6. キャリアパス階層（上位X%ごとの到達点）

### 昇進ピラミッド

```
                    ┌──────────┐
                    │  社長・会長 │  ← 同期の約0.5%未満
                    └────┬─────┘
                   ┌─────┴──────┐
                   │  副社長・専務 │  ← 上位約1〜2%
                   └─────┬──────┘
                  ┌──────┴───────┐
                  │ 執行役員(T)   │  ← 上位約3〜5%
                  └──────┬───────┘
              ┌──────────┴──────────┐
              │   部長(S)          │  ← 上位約10〜20%
              └──────────┬──────────┘
          ┌──────────────┴──────────────┐
          │   課長(M4〜M5)             │  ← 上位約40〜50%
          └──────────────┬──────────────┘
      ┌──────────────────┴──────────────────┐
      │    管理職(M1〜M3)                │  ← 大多数が到達
      └────────────────────────────────────┘
```

### 到達点の目安

| レベル | 割合（概算） | 典型的なキャリア |
|--------|-------------|----------------|
| **トップ0.5%** | 約1名/年 | 社長候補。複数部門長を歴任 |
| **上位1〜3%** | 2〜4名/年 | 執行役員（Tグレード）。部門長や海外統括 |
| **上位5〜10%** | 5〜10名/年 | 部長（Sグレード）。事業の責任者 |
| **上位10〜20%** | 10〜20名/年 | 子会社社長・取締役 |
| **上位20〜50%** | 20〜50名/年 | 課長（M4〜M5）。実務のスペシャリスト |
| **上位50〜100%** | — | 関連会社・転職 |

### 子会社社長ポスト

- グループ会社**約498社**: 子会社社長ポストは非常に豊富
- 電力事業会社の社長ポストが特に多い（20カ国以上）
- Helena（米国農業）の幹部ポスト
- 少数精鋭のため、一人当たりのチャンスは比較的多い
- 部長・課長クラスで子会社の社長・役員に出向するのが一般的
- 本社の課長職に占めるキャリア採用者の割合: 約2割

- 「グループCEO」廃止、「統括役員」新設

### 中期経営戦略「GC2027」

- 戦略プラットフォーム型事業の5分野に注力
