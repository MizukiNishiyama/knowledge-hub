> 「堅実の住友」。住友財閥の「信用を重んじ、浮利を追わず」の精神を継承。メディア・不動産・リース・電力など非資源事業に強み。五大商社の中で最も堅実・保守的な社風。

## 2. 組織構造・階層

### 事業部門制（ビジネスユニット制）

住友商事は **6つの事業部門** を中心に組織を構成。

```
社長・CEO
├── 金属事業部門
├── 輸送機・建機事業部門
├── インフラ事業部門
├── メディア・デジタル事業部門
├── 生活・不動産事業部門
├── 資源・化学品事業部門
├── エネルギーイノベーション・イニシアチブ
└── コーポレートグループ
    ├── 経営企画部
    ├── 人事厚生部
    ├── 経理部
    ├── 法務部
    ├── リスクマネジメント部
    ├── IT推進部
    └── サステナビリティ推進部
```

### 職位階層

```
会長 → 社長 → 副社長 → 専務執行役員
→ 常務執行役員 → 執行役員
→ 事業部門長（GM） → 部長 → チーム長（課長）
→ 担当
```

## 5. 新卒入社後のキャリアパス

### 入社〜5年目（若手期）

```
入社 → 配属先決定（6事業部門 or コーポレート）
  → 1〜2年目: OJT中心。トレーディング・事業管理
  → 3年目前後: 海外研修制度（約1年間）
  → 4〜5年目: 担当案件の主力に
```

- **新卒採用数**: 毎年約130〜150名（総合職）
- **社風**: 堅実で温かみのある雰囲気。「住友の結束」
- **ワークライフバランス**: 五大商社の中で比較的良好との評価

### 6年目〜15年目（中堅期）

- **海外駐在**: 約50〜60%が経験
  - 主要駐在先: 東南アジア、北米、欧州、アフリカ
  - 期間: 3〜5年
- **子会社出向**: SCSK、J:COM、サミットなど
- **昇進**: チーム長/課長（30代後半〜40代前半）

### 15年目以降（管理職期）

- **部長**: 40代半ば〜
- **事業部門長**: 50代前半〜
- **執行役員**: 50代半ば〜

## 6. キャリアパス階層（上位X%ごとの到達点）

### 昇進ピラミッド

```
                    ┌──────────┐
                    │  社長・会長 │  ← 同期の約0.5%未満
                    └────┬─────┘
                   ┌─────┴──────┐
                   │  副社長・専務 │  ← 上位約1〜2%
                   └─────┬──────┘
                  ┌──────┴───────┐
                  │   執行役員    │  ← 上位約3〜5%
                  └──────┬───────┘
              ┌──────────┴──────────┐
              │   部長・事業部門長   │  ← 上位約10〜15%
              └──────────┬──────────┘
          ┌──────────────┴──────────────┐
          │    チーム長（課長級）       │  ← 上位約35〜45%
          └──────────────┬──────────────┘
      ┌──────────────────┴──────────────────┐
      │          担当                     │  ← 残り
      └────────────────────────────────────┘
```

### 到達点の目安

| レベル | 割合（概算） | 典型的なキャリア |
|--------|-------------|----------------|
| **トップ0.5%** | 約1名/年 | 社長候補。事業部門長を複数経験 |
| **上位1〜3%** | 3〜5名/年 | 執行役員。事業部門長や海外地域統括 |
| **上位5〜10%** | 7〜13名/年 | 部長級。部門内の責任者 |
| **上位10〜20%** | 13〜25名/年 | 子会社社長・取締役 |
| **上位20〜45%** | 25〜60名/年 | チーム長。専門分野を深耕 |
| **上位45〜100%** | — | 関連会社・転職 |

### 子会社社長ポスト

- 連結子会社約330社、持分法適用約200社
- SCSK社長は住友商事出身者が就任するケースが多い
- J:COMはKDDIとの共同運営のためボードメンバーとして参画
- 堅実な社風のため、長期的にポストを得る傾向

## 7. 住友商事の強み・特徴

### 「堅実の住友」
- 住友家の家訓「浮利を追わず」の精神
- リスク管理が厳格で、大きな損失を出しにくい体質
- 2014年のシェールオイル減損（約1,700億円）を教訓に体質改善

### メディア・デジタル分野の強さ
- SCSK: 安定高収益のSIer。DX需要で成長
- J:COM: 約570万世帯の顧客基盤
- T-Gaia: 携帯販売のインフラ

### 非資源事業への注力
- 資源依存度が低く（約30%）、安定した収益
- 不動産・リテール・物流など内需型事業が充実

### グローバルなインフラ事業
- 海外電力事業（IPP）
- 通信インフラ（ミャンマーなど新興国）
