> 「堅実の住友」。住友財閥の「信用を重んじ、浮利を追わず」の精神を継承。メディア・不動産・リース・電力など非資源事業に強み。2024年4月に6事業部門制から9グループ制へ大規模組織再編。

## 2. 組織構造・階層

### 9グループ制（2024年4月〜、SBUベース）

住友商事は **2024年4月1日付** で大規模な組織再編を実施。従来の6事業部門制から **9つの営業グループ** へ移行。戦略事業単位（Strategic Business Unit: SBU）をベースに、戦略上の親和性が高いSBUを束ねる形で編成。

```
社長・CEO
├── 1. 鉄鋼グループ（鉄鋼製品のグローバルサプライチェーン、鉄鋼GX）
├── 2. 自動車グループ（自動車トレード、モビリティサービス）
├── 3. 輸送機・建機グループ（航空宇宙、船舶、建設機械レンタル）
├── 4. 都市総合開発グループ（不動産開発、スマートシティ、物流施設）
├── 5. メディア・デジタルグループ（SCSK、通信事業、デジタルソリューション）
├── 6. ライフスタイルグループ（食品、リテール、ヘルスケア）
├── 7. 資源グループ（非鉄金属、石炭、鉄鉱石等の資源権益）
├── 8. 化学品・エレクトロニクス・農業グループ（合成樹脂、電子材料、農業資材）
├── 9. エネルギートランスフォーメーショングループ（再エネ、水素・アンモニア、電力インフラ）
└── コーポレートグループ（6グループに再編）
    ├── 経営企画部
    ├── 人事厚生部
    ├── 経理部
    ├── 法務部
    ├── リスクマネジメント部
    ├── IT推進部
    └── サステナビリティ推進部
```

**組織再編のポイント**: 「エネルギートランスフォーメーショングループ」と「都市総合開発グループ」は従来の商品軸ではなく **戦略軸** で編成された新しいタイプの組織。

### 職位階層

```
会長 → 社長 → 副社長 → 専務執行役員
→ 常務執行役員 → 執行役員
→ グループ長 → 部長 → チーム長（課長）
→ 担当
```

## 6. キャリアパス階層（上位X%ごとの到達点）

### 昇進ピラミッド

```
                    ┌──────────┐
                    │  社長・会長 │  ← 同期の約0.5%未満
                    └────┬─────┘
                   ┌─────┴──────┐
                   │  副社長・専務 │  ← 上位約1〜2%
                   └─────┬──────┘
                  ┌──────┴───────┐
                  │   執行役員    │  ← 上位約3〜5%
                  └──────┬───────┘
              ┌──────────┴──────────┐
              │   部長・グループ長   │  ← 上位約10〜15%
              └──────────┬──────────┘
          ┌──────────────┴──────────────┐
          │    課長級（AP4）           │  ← 上位約50%
          └──────────────┬──────────────┘
      ┌──────────────────┴──────────────────┐
      │    係長級（AP5）以下             │  ← 残り
      └────────────────────────────────────┘
```

### 到達点の目安

| レベル | 割合（概算） | 典型的なキャリア |
|--------|-------------|----------------|
| **トップ0.5%** | 約1名/年 | 社長候補。グループ長を複数経験 |
| **上位1〜3%** | 3〜5名/年 | 執行役員。グループ長や海外地域統括 |
| **上位5〜10%** | 7〜13名/年 | 部長級（AP3以上）。グループ内の責任者 |
| **上位10〜20%** | 13〜25名/年 | 子会社社長・取締役 |
| **上位20〜50%** | 25〜60名/年 | 課長級（AP4）。専門分野を深耕 |
| **上位50〜100%** | — | 関連会社経営層として出向・転職 |

### 昇進の分岐点
- **入社10年目前後**: AP（管理職）への昇格タイミングで最初の差がつく。ここまでは年功序列的に横並び
- **40代前半**: 課長級（AP4）に昇格できるかが大きな分岐点。ここで**約半数がふるいにかけられる**
- **40代後半〜**: 部長級（AP3）以上に進めるのは**同期の約20%以下**。それ以外は関連会社の経営層として出向

### 子会社社長ポスト

- 連結子会社・関連会社を数百社擁しており、子会社社長ポストは相当数存在
- 40代以降、本体での部長昇進が難しい場合、**関連会社の経営層（取締役・社長）への出向**が一般的
- 若手（30代）でも小規模な関連会社であれば**役員として出向**するケースあり
- SCSK社長は住友商事出身者が就任するケースが多い
- J:COMはKDDIとの共同運営のためボードメンバーとして参画

### 出世コースの特徴
- **海外駐在経験は事実上必須**: 出世ラインに乗るためには海外駐在の実績が重要
- **新規ビジネスの立ち上げ（0→1）**: 商社で最も高く評価される
- **経営企画部の部長**: 典型的な出世コースの一つ → グループ長 → 役員へ

## 7. 住友商事の強み・特徴

### 「堅実の住友」
- 住友家の家訓「浮利を追わず」の精神
- リスク管理が厳格で、大きな損失を出しにくい体質
- 2014年のシェールオイル減損（約1,700億円）を教訓に体質改善

### メディア・デジタル分野の強さ
- SCSK: 安定高収益のSIer。DX需要で成長
- J:COM: 約570万世帯の顧客基盤
- T-Gaia: 携帯販売のインフラ

### 2024年の大規模組織再編
- 6事業部門制 → 9グループ制への移行
- SBUベースの戦略的組織編成
- 「エネルギートランスフォーメーション」「都市総合開発」という戦略軸グループの新設

### 非資源事業への注力
- 非資源利益比率約70%超の安定した収益構造
- 不動産・リテール・物流など内需型事業が充実

### グローバルなインフラ事業
- 海外電力事業（IPP）
- 通信インフラ（新興国中心）
- 再エネ・水素・アンモニアへのエネルギートランジション
