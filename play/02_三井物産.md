# 三井物産（Mitsui & Co.）

> 「人の三井」と称される。資源・エネルギー分野に圧倒的な強みを持ち、LNG取扱量は世界トップクラス。個人の裁量と自由度を重視する社風。

## 2. 組織構造・階層

### 本部制（ビジネスユニット制）

三井物産は **16の事業本部** を中心に組織編成。

```
社長・CEO
├── 鉄鋼製品本部
├── 金属資源本部
├── エネルギー第一本部（石油・ガス上流）
├── エネルギー第二本部（LNG・ガストレーディング）
├── プロジェクト本部（インフラ・プラント）
├── モビリティ第一本部
├── モビリティ第二本部
├── 化学品本部
├── 食料本部
├── 流通事業本部
├── ウェルネス事業本部（ヘルスケア）
├── ニュートリション・アグリカルチャー本部
├── ICT事業本部
├── コーポレートディベロップメント本部
├── エネルギーソリューション本部
└── コーポレートスタッフ
    ├── 経営企画部
    ├── 人事総務部
    ├── 経理部
    ├── 法務部
    └── リスクマネジメント部
```

### 職位階層

```
会長 → 社長 → 副社長 → 専務執行役員
→ 常務執行役員 → 執行役員
→ 本部長 → 部長 → 室長 → 課長 → 担当
```

## 5. 新卒入社後のキャリアパス

### 入社〜5年目（若手期）

```
入社 → 配属先決定（事業本部 or コーポレート）
  → 1〜2年目: OJT、トレーディング/事業管理の基本
  → 3年目前後: 海外トレーニー制度（約1年間の海外派遣）
  → 4〜5年目: 主担当として案件推進
```

- **新卒採用数**: 毎年約130〜150名（担当職＝総合職）
- **「人の三井」**: 個人の裁量が大きく、若手から大きな案件を任される傾向
- **海外トレーニー**: 全員ではないが高い割合で海外研修を経験

### 6年目〜15年目（中堅期）

- **海外駐在**: 約60〜70%が経験。三井物産は海外拠点が約130拠点
  - 主要駐在先: 北米、豪州、東南アジア、ブラジル、中東、アフリカ
  - 期間: 3〜5年
- **子会社出向**: 事業投資先の経営への参画
  - IHH Healthcare、各エネルギー事業会社、食料関連事業会社
- **昇進**: 課長クラス（30代後半〜40代前半）

### 15年目以降（管理職期）

- **部長**: 40代半ば〜（選抜が本格化）
- **本部長**: 50代前半〜（事業の責任者）
- **執行役員**: 50代半ば〜

## 6. キャリアパス階層（上位X%ごとの到達点）

### 昇進ピラミッド

```
                    ┌──────────┐
                    │  社長・会長 │  ← 同期の約0.5%未満
                    └────┬─────┘
                   ┌─────┴──────┐
                   │  副社長・専務 │  ← 上位約1〜2%
                   └─────┬──────┘
                  ┌──────┴───────┐
                  │   執行役員    │  ← 上位約3〜5%
                  └──────┬───────┘
              ┌──────────┴──────────┐
              │   本部長・部長級    │  ← 上位約10〜15%
              └──────────┬──────────┘
          ┌──────────────┴──────────────┐
          │        課長級              │  ← 上位約30〜40%
          └──────────────┬──────────────┘
      ┌──────────────────┴──────────────────┐
      │        担当（シニアスタッフ）       │  ← 大多数
      └────────────────────────────────────┘
```

### 到達点の目安

| レベル | 割合（概算） | 典型的なキャリア |
|--------|-------------|----------------|
| **トップ0.5%** | 約1名/年 | 社長候補。複数本部長を歴任、海外拠点長経験あり |
| **上位1〜3%** | 3〜5名/年 | 執行役員。主要事業本部長や海外地域統括 |
| **上位5〜10%** | 7〜13名/年 | 部長級。事業本部内の部門責任者 |
| **上位10〜20%** | 13〜25名/年 | 子会社社長・取締役。投資先のマネジメント |
| **上位20〜40%** | 25〜50名/年 | 課長クラス。専門分野で活躍 |
| **上位40〜100%** | — | 関連会社転籍・転職・独立 |

### 子会社社長ポスト

- 連結子会社約280社、持分法適用会社約200社
- 子会社社長ポストは部長級以上（上位15%前後）で到達
- 大型事業会社（IHH、パンサ・エナジー等）のボードメンバーは執行役員級
- 「人の三井」の文化で、独立起業する卒業生も多い

## 7. 三井物産の強み・特徴

### 「人の三井」
- 個人の裁量が大きく、起業家精神を持つ人材が多い
- OBネットワークが強力（「三井物産マフィア」）
- 独立・起業して活躍するOBが多い

### 資源・エネルギーの圧倒的強さ
- LNG: 世界のLNG供給量の約5%に関与（生産・トレーディング含む）
- 鉄鉱石: 豪州・ブラジルの権益
- 原油・ガス: 世界各地の上流権益

### ヘルスケア事業の先行者利益
- IHH Healthcare（時価総額約3兆円）の大株主
- アジア最大級の病院ネットワーク
- 商社の中でヘルスケア分野への参入が最も進んでいる

### 次世代事業への投資
- モビリティ（CaaS/MaaS）
- 再生可能エネルギー
- DX・デジタル分野
