# 三井物産（Mitsui & Co.）

> 「人の三井」と称される。資源・エネルギー分野に圧倒的な強みを持ち、LNG取扱量は世界トップクラス。個人の裁量と自由度を重視する社風。2024年7月に業務職を廃止し総合職に一本化。

---

## 2. 組織構造・階層

### 本部制（16事業本部＋7セグメント）

三井物産は **16の事業本部** と **7つのオペレーティングセグメント** で構成（2025年10月時点）。

```
社長・CEO
├── 金属資源本部（鉄鉱石、原料炭、銅等の開発・加工・販売）
├── エネルギー第一本部（資源採掘・開発、エネルギー資源の輸入・販売）
├── エネルギー第二本部（LNG事業開発、供給販売、物流取引）
├── エネルギーソリューション本部（再エネ、水素、アンモニア等の次世代エネルギー）
├── プロジェクト本部（電力、資源インフラ、社会環境インフラ）
├── モビリティ第一本部（自動車、建設機械、鉄道）
├── モビリティ第二本部（船舶、航空機、宇宙関連）
├── ベーシックマテリアルズ本部（メタノール、化学品、タンクターミナル）
├── パフォーマンスマテリアルズ本部（合成樹脂、先端材料）
├── ニュートリション・アグリカルチャー本部（農薬、動物薬、機能性素材）
├── 鉄鋼製品本部（鉄鋼製品製造・加工、リサイクル）
├── 食料本部（糖質・脂質・たんぱく質のグローバル物流・事業）
├── 流通事業本部（中間流通、ブランド&リテール、食品製造）
├── ウェルネス事業本部（医療、ファーマ）
├── ICT事業本部（デジタルソリューション、消費者プラットフォーム）
├── コーポレートディベロップメント本部（金融、不動産、物流、戦略的案件）
├── 地域本部
│   ├── 米州本部
│   └── アジア・大洋州本部
└── コーポレートスタッフ
    ├── 経営企画部
    ├── 人事総務部
    ├── 経理部
    ├── 財務部
    ├── 法務部
    └── リスクマネジメント部
```

### 職位階層

```
会長 → 社長 → 副社長執行役員（CHRO・CCO、CFO等）
→ 専務執行役員 → 常務執行役員 → 執行役員
→ 本部長 → 部長 → 室長 → 課長 → 担当
```

### 職群（等級）の構成

```
担当職1級（1号→2号→3号、毎年自動昇格）
→ 担当職SF（シニアフェロー）
→ M3（管理職入口）→ M2（課長級）→ M1（部長級）
```

管理職の3区分:
- **MT（マネジメント群）**: 部店長、海外店長など
- **MS（マネジリアルスタッフ群）**: 本部長補佐、大規模海外店長補佐
- **SS（シニアスタッフ群）**: 専門性を活かした上級スタッフ

## 5. 新卒入社後のキャリアパス

### 役職グレードと年収レンジ

| グレード | 役職 | 目安年次 | 年収目安 |
|---------|------|---------|---------|
| 担当職1級（1〜3号） | 担当 | 1〜3年目 | 400〜700万円 |
| 担当職SF | 担当（上位） | 4〜6年目 | 700〜1,000万円 |
| M3 | 管理職入口 | 7〜9年目 | 1,200〜1,600万円 |
| M2 | 課長級 | 10年目〜 | 1,600〜2,500万円 |
| M1 | 部長級 | 15〜20年目 | 2,500〜4,000万円 |

※全社平均年収: **1,996万円**（2025年3月期）

### 入社〜6年目（若手期・担当職1級→SF）

```
入社 → 配属先決定（事業本部 or コーポレート）
  → 1〜3年目: 担当職1級（毎年自動昇格 1号→2号→3号）
  → 4年目: 担当職SFに昇進
  → 5〜6年目: 30歳前後で年収約1,000万円到達
```

- **新卒採用数**: 毎年約130〜150名（総合職）
- **「人の三井」**: 個人の裁量が大きく、若手から大きな案件を任される傾向
- **海外トレーニー**: 全員ではないが高い割合で海外研修を経験
- **2024年7月制度改革**: 業務職を廃止し総合職に一本化。「Global」（全国・海外）と「Regional」（採用地域限定）の勤務地選択制を導入
- **中途採用比率**: 約41.4%（2024年度）

### 7年目〜15年目（中堅期・M3→M2）

- **M3昇格**: 入社7〜9年目に管理職昇進試験（ほぼ全員合格）
- **海外駐在**: 約60〜70%が経験。海外拠点は111拠点
  - 主要駐在先: 北米、豪州、東南アジア、ブラジル、中東、アフリカ
  - 期間: 3〜5年
  - 駐在時は手当が手厚く、**日本の額面年収2,000〜2,500万円相当の手取り**が30代でも可能
- **子会社出向**: 事業投資先の経営への参画
  - IHH Healthcare、各エネルギー事業会社、食料関連事業会社
- **M2昇格**: ここから明確に差がつき始める（年収1,600〜2,500万円）

### 15年目以降（管理職期・M1〜）

- **M1（部長級）**: 年収2,500〜4,000万円。競争が激化
- **本部長**: 50代前半〜（事業の責任者）
- **執行役員**: 入社から部長までの平均年数は **約27年（平均49歳）**

## 6. キャリアパス階層（上位X%ごとの到達点）

### 昇進ピラミッド

```
                    ┌──────────┐
                    │  社長・会長 │  ← 同期の約0.5%未満
                    └────┬─────┘
                   ┌─────┴──────┐
                   │  副社長・専務 │  ← 上位約1〜2%
                   └─────┬──────┘
                  ┌──────┴───────┐
                  │   執行役員    │  ← 上位約3〜5%
                  └──────┬───────┘
              ┌──────────┴──────────┐
              │   本部長・部長(M1)  │  ← 上位約10〜15%
              └──────────┬──────────┘
          ┌──────────────┴──────────────┐
          │      課長級(M2)           │  ← 大多数
          └──────────────┬──────────────┘
      ┌──────────────────┴──────────────────┐
      │    管理職入口(M3) ← ほぼ全員到達  │
      └────────────────────────────────────┘
```

### 到達点の目安

| レベル | 割合（概算） | 典型的なキャリア |
|--------|-------------|----------------|
| **トップ0.5%** | 約1名/年 | 社長候補。複数本部長を歴任、海外拠点長経験あり |
| **上位1〜3%** | 3〜5名/年 | 執行役員。主要事業本部長や海外地域統括 |
| **上位5〜10%** | 7〜13名/年 | 部長級（M1）。事業本部内の部門責任者 |
| **上位10〜20%** | 13〜25名/年 | 子会社社長・取締役。投資先のマネジメント |
| **上位20〜40%** | 25〜50名/年 | 課長クラス（M2）。専門分野で活躍 |
| **上位40〜100%** | — | 関連会社転籍・転職・独立 |

### 昇進の分岐点
- **M3（管理職）まで**: 入社8〜9年目。試験はあるが**ほぼ全員通過**
- **M2（課長級）**: 大多数が到達。ここから徐々に差がつく
- **M1（部長級）**: **競争が激化**。年収3,000〜4,000万円
- **本部長・執行役員**: 入社から部長まで平均27年（12名中8名が入社25〜27年で部長到達）
- **取締役**: 役員数19名（うち女性4名、女性比率21.1%）

### 子会社社長ポスト

- 連結子会社280社（国内77社・海外203社）、持分法適用147社
- 子会社社長ポストは部長級（M1）以上で到達
- **40歳前後で子会社の社長**になる可能性もある
- 大型事業会社（IHH、パンサ・エナジー等）のボードメンバーは執行役員級
- 「人の三井」の文化で、**独立起業する卒業生も多い**

### 評価制度
- **貢献度評価**（ボーナスに反映）と**個人能力評価**（昇給に反映）の2軸
- 個人能力評価は5段階で、過去3年間の平均で昇給額が決定
- 管理職には**360度評価**を実施
- **コンピテンシー評価**も導入

## 7. 三井物産の強み・特徴

### 「人の三井」
- 個人の裁量が大きく、起業家精神を持つ人材が多い
- OBネットワークが強力（「三井物産マフィア」）
- 独立・起業して活躍するOBが多い

### 資源・エネルギーの圧倒的強さ
- LNG: 世界のLNG供給量の約5%に関与（生産・トレーディング含む）
- 鉄鉱石: 豪州・ブラジルの権益
- 原油・ガス: 世界各地の上流権益
- 金属資源セグメント利益: 2,854億円（セグメント最大）

### ヘルスケア事業の先行者利益
- IHH Healthcare（時価総額約3兆円）の大株主
- アジア最大級の病院ネットワーク
- 商社の中でヘルスケア分野への参入が最も進んでいる

### 次世代事業への投資
- モビリティ（CaaS/MaaS）
- 再生可能エネルギー・水素・アンモニア（エネルギーソリューション本部）
- DX・デジタル分野

### 2025年の主な社名変更
- 三井石油開発 → **三井エネルギー資源開発**（2025年1月）
- 三井物産プラントシステム → **三井物産プロジェクトソリューション**（2024年10月）
- DM三井製糖ホールディングス → **DM三井製糖**（2025年4月）
