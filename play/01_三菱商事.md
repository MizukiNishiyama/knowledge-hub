# 三菱商事（Mitsubishi Corporation）

> 五大商社の盟主。時価総額・純利益で長年トップを維持。「組織の三菱」と称される。

---

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| 設立 | 1950年（前身は1870年の九十九商会） |
| 本社 | 東京都千代田区丸の内 |
| 従業員数 | 単体 約5,400名 / 連結 約80,000名 |
| 時価総額 | 約10兆円超（2025年3月期時点） |
| 三菱グループ | 三菱グループの中核企業（三菱UFJ銀行・三菱重工業と並ぶ「御三家」） |

## 2. 組織構造・階層

### カンパニー制（事業グループ制）

三菱商事は **10の事業グループ** を中心とした組織体制を採用。

```
社長・CEO
├── 天然ガスグループ
├── 総合素材グループ
├── 石油・化学グループ
├── 金属資源グループ
├── 産業インフラグループ
├── 自動車・モビリティグループ
├── 食品産業グループ
├── コンシューマー産業グループ
├── 電力ソリューショングループ
├── 複合都市開発グループ
└── コーポレートスタッフ部門
    ├── 経営企画部
    ├── 人事部
    ├── 財務部
    ├── 法務部
    ├── リスクマネジメント部
    └── IT部門
```

### 階層構造（職位）

```
会長 → 社長 → 副社長（複数） → 常務執行役員
→ 執行役員 → グループCEO → 本部長
→ 部長 → 課長 → 課長代理 → 担当者
```

## 5. 新卒入社後のキャリアパス

### 入社〜5年目（若手期）

```
入社 → 配属先決定（希望＋適性）
  → 1〜2年目: OJT中心、トレーディング実務・事業管理
  → 3〜4年目: 担当業務の主力に（単独での案件推進）
  → 4〜5年目: 海外駐在 or 子会社出向の打診
```

- **新卒採用数**: 毎年約150〜170名（総合職）
- **配属**: 10事業グループ＋コーポレートスタッフ部門のいずれか
- **語学研修**: 入社2年目前後で海外語学研修（1年程度）の機会あり

### 6年目〜15年目（中堅期）

- **海外駐在**: 約60〜70%が一度は海外駐在を経験
  - 駐在先: 北米、欧州、東南アジア、豪州、中東、アフリカなど
  - 期間: 通常3〜5年
- **子会社出向**: 事業投資先への出向（経営参画）
  - ローソン、三菱食品、Eneco、各海外事業会社 など
- **昇進**: 課長代理 → 課長（30代後半〜40代前半）

### 15年目以降（管理職期）

- **部長**: 40代半ば〜後半（選抜組）
- **グループ内異動**: 複数の事業グループを経験するケースも
- **本部長・執行役員**: 50代前半〜（トップ層）

## 6. キャリアパス階層（上位X%ごとの到達点）

### 昇進ピラミッド

```
                    ┌──────────┐
                    │  社長・会長 │  ← 同期入社者の約0.5%未満
                    └────┬─────┘
                   ┌─────┴──────┐
                   │  副社長・常務 │  ← 上位約1〜2%
                   └─────┬──────┘
                  ┌──────┴───────┐
                  │   執行役員    │  ← 上位約3〜5%
                  └──────┬───────┘
              ┌──────────┴──────────┐
              │   部長・本部長級    │  ← 上位約10〜15%
              └──────────┬──────────┘
          ┌──────────────┴──────────────┐
          │     課長（マネージャー）     │  ← 上位約40〜50%
          └──────────────┬──────────────┘
      ┌──────────────────┴──────────────────┐
      │    課長代理・担当課長              │  ← 大多数が到達
      └────────────────────────────────────┘
```

### キャリアの選択肢

| レベル | 割合（概算） | 典型的なキャリア |
|--------|-------------|----------------|
| **トップ0.5%** | 約1名/年 | 社長・CEO候補。複数の事業グループCEOを歴任 |
| **上位1〜3%** | 3〜5名/年 | 執行役員・常務。事業グループCEOや海外地域統括 |
| **上位5〜10%** | 8〜15名/年 | 部長・本部長。特定事業の責任者 |
| **上位10〜20%** | 15〜30名/年 | 子会社社長・取締役。事業投資先のトップマネジメント |
| **上位20〜40%** | 30〜60名/年 | 課長クラスで専門領域を深耕。海外駐在複数回 |
| **上位40〜70%** | 60〜100名/年 | 課長〜課長代理。実務のスペシャリスト |
| **上位70〜100%** | — | 関連会社への転籍・独立・転職 |

### 子会社社長ポスト

- 三菱商事グループには **連結子会社約700社** が存在
- 子会社社長ポストは多数あり、部長級以上（上位10〜20%）で到達可能
- 大型子会社（ローソン、三菱食品等）の社長は執行役員級
- 海外現地法人の社長は部長〜本部長級が就任

## 7. 三菱商事の強み・特徴

### 「組織の三菱」
- 三菱グループ（三菱UFJ、三菱重工、AGC、キリン等）との強固なネットワーク
- グループ全体での事業創出力が他商社を圧倒
- 財閥系の組織力と安定性

### 資源分野の圧倒的強さ
- LNG: 世界最大級の取扱量
- 原料炭: 豪州BMAジョイントベンチャー（BHP との50:50JV）
- 銅: ペルーQuellaveco鉱山の本格稼働

### 非資源分野への多角化
- ローソン（リテール）、三菱食品（食品バリューチェーン）
- Eneco（再エネ）、都市開発事業
- 資源依存からの脱却を着実に推進
