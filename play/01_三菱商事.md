> 五大商社の盟主。時価総額・純利益で長年トップを維持。「組織の三菱」と称される。

## 2. 組織構造・階層

### グループ制（8営業グループ体制）

三菱商事は **8つの営業グループ** を中心とした組織体制を採用（2025年12月時点）。

```
社長・CEO
├── 地球環境エネルギーグループ（LNG・天然ガス開発、次世代エネルギー）
├── 金属資源グループ（原料炭、銅、鉄鉱石等）
├── マテリアルソリューショングループ（化学品、鉄鋼製品等）
├── 社会インフラグループ（プラント、産業機械、船舶等）
├── モビリティグループ（自動車関連事業）
├── 食品産業グループ（食品流通、食品製造）
├── S.L.C.グループ（スマートライフクリエーション：リテール、ヘルスケア、生活消費財）
├── 電力ソリューショングループ（発電事業、電力トレーディング）
└── コーポレートスタッフ部門
    ├── 経営企画部
    ├── 人事部
    ├── 財務部
    ├── 法務部
    ├── リスクマネジメント部
    └── AI・IT戦略部（2026年度に「ITサービス部」から改称）
```

### 2026年度の主な組織変更
- **AI戦略担当**を新設
- モビリティサービス本部とeモビリティソリューション本部を「**モビリティ・ソリューション本部**」に統合
- グループ会社数: **約1,700社**（子会社約1,265社＋関連会社約456社）

### 階層構造（職位）

```
会長 → 社長 → 副社長（複数） → 常務執行役員
→ 執行役員 → グループCEO → 本部長
→ 部長 → 課長 → 課長代理 → 担当者
```

## 5. 新卒入社後のキャリアパス

### 役職グレードと年収レンジ

| グレード | 役職 | 年収目安 | 到達年齢の目安 |
|---------|------|---------|-------------|
| S2 | 担当 | 550〜900万円 | 入社1〜3年目 |
| S1 | 担当（上位） | 1,000〜1,400万円 | 20代後半 |
| AS | 主任級 | 1,500〜2,000万円 | 30代前半 |
| M2 | マネージャー（課長） | 2,000〜2,600万円 | 30代後半 |
| M1 | 部長代行 | 2,600万円以上 | 40代前半 |
| MP | 部長 | 3,000万円以上 | 40代半ば |

※全社平均年収: **約2,090万円**（有価証券報告書ベース、五大商社トップ）

### 入社〜5年目（若手期・グレードS2→S1）

```
入社 → 配属先決定（希望＋適性）
  → 1〜2年目: OJT中心、トレーディング実務・事業管理
  → 約2割が入社直後に子会社・グループ会社に出向（3〜5年）
  → 3〜4年目: 担当業務の主力に（単独での案件推進）
  → 4〜5年目: 海外駐在 or 子会社出向の打診
```

- **新卒採用数**: 毎年約150〜170名（総合職）
- **配属**: 8営業グループ＋コーポレートスタッフ部門のいずれか
- **語学研修**: 入社2年目前後で海外語学研修（1年程度）の機会あり
- **出向の目的**: 「現地・現物・現場」でモノとお金の流れを体感させること

### 6年目〜15年目（中堅期）

- **海外駐在**: 約60〜70%が一度は海外駐在を経験
  - 駐在先: 北米、欧州、東南アジア、豪州、中東、アフリカなど
  - 期間: 通常3〜5年
- **子会社出向**: 事業投資先への出向（経営参画）
  - ローソン、三菱食品、Eneco、各海外事業会社 など
- **昇進**: 課長代理 → 課長（30代後半〜40代前半）

### 15年目以降（管理職期）

- **部長**: 40代半ば〜後半（選抜組）
- **グループ内異動**: 複数の事業グループを経験するケースも
- **本部長・執行役員**: 50代前半〜（トップ層）

## 6. キャリアパス階層（上位X%ごとの到達点）

### 昇進ピラミッド

```
                    ┌──────────┐
                    │  社長・会長 │  ← 同期入社者の約0.5%未満
                    └────┬─────┘
                   ┌─────┴──────┐
                   │  副社長・常務 │  ← 上位約1〜2%
                   └─────┬──────┘
                  ┌──────┴───────┐
                  │   執行役員    │  ← 上位約3〜5%
                  └──────┬───────┘
              ┌──────────┴──────────┐
              │   部長・本部長級    │  ← 上位約10〜15%
              └──────────┬──────────┘
          ┌──────────────┴──────────────┐
          │     課長（マネージャー）     │  ← 上位約40〜50%
          └──────────────┬──────────────┘
      ┌──────────────────┴──────────────────┐
      │    課長代理・担当課長              │  ← 大多数が到達
      └────────────────────────────────────┘
```

### キャリアの選択肢

| レベル | 割合（概算） | 典型的なキャリア |
|--------|-------------|----------------|
| **トップ0.5%** | 約1名/年 | 社長・CEO候補。複数の事業グループCEOを歴任 |
| **上位1〜3%** | 3〜5名/年 | 執行役員・常務。事業グループCEOや海外地域統括 |
| **上位5〜10%** | 8〜15名/年 | 部長・本部長。特定事業の責任者 |
| **上位10〜20%** | 15〜30名/年 | 子会社社長・取締役。事業投資先のトップマネジメント |
| **上位20〜40%** | 30〜60名/年 | 課長クラスで専門領域を深耕。海外駐在複数回 |
| **上位40〜70%** | 60〜100名/年 | 課長〜課長代理。実務のスペシャリスト |
| **上位70〜100%** | — | 関連会社への転籍・独立・転職 |

### 子会社社長ポスト

- 三菱商事グループには **約1,700社**（子会社約1,265社＋関連会社約456社）が存在
- 子会社社長ポストは非常に多く、部長級以上（上位10〜20%）で到達可能
- 大型子会社（三菱食品、メタルワン等）の社長は執行役員級
- 海外現地法人の社長は部長〜本部長級が就任
- **2027年度からAI関連資格が課長級（M2）昇格の要件に追加**

### 昇進の特徴
- **マネージャー（M2）まで**: 基本的に同期全員がほぼ横並びで昇進（年功序列色が強い）
- **マネージャー以降**: 成果によって差が出始め、出世コースに乗れるのは**同期の約半数**
- **部長クラス（MP）**: 到達できるのは**同期の約20%以下**
- **40代が分岐点**: 関連会社の重役・経営層として出向する割合が増加

## 7. 三菱商事の強み・特徴

### 「組織の三菱」
- 三菱グループ（三菱UFJ、三菱重工、AGC、キリン等）との強固なネットワーク
- グループ全体での事業創出力が他商社を圧倒
- 財閥系の組織力と安定性

### 資源分野の圧倒的強さ
- LNG: 世界最大級の取扱量
- 原料炭: 豪州BMAジョイントベンチャー（BHP との50:50JV）
- 銅: ペルーQuellaveco鉱山の本格稼働

### 非資源分野への多角化
- ローソン（リテール）、三菱食品（食品バリューチェーン）
- Eneco（再エネ）、都市開発事業
- 資源依存からの脱却を着実に推進
