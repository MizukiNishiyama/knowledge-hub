## 論点2: デジタル庁外部人材の採用基準・選考プロセス——何が評価軸か

### 公式採用要件（確認済み職種）

#### データサイエンティスト（AI利活用）職
- **必須スキル**:
  - 検索システムの開発・改善経験
  - フリーテキスト・Excel・PDFのデータクレンジング対応力
  - 機械学習・自然言語処理の研究知識と数理的バックグラウンド
  - 学術的アウトプット経験
  - AIエシックス・責任あるAIの理解
  - 行政法・政策のドメイン知識（尚可）
  - 調達仕様書の作成・ベンダーコミュニケーション経験
- **選考プロセス**: 書類選考 → 2〜3回の面接（オンライン原則） → 内定（約4〜6週間）
- **ソース**: [HERPデジタル庁求人 データサイエンティスト](https://herp.careers/v1/digitalsaiyo/3Ja4Dbj4ehPC)

#### シニアエキスパート（新規事業）職（過去公募）
- **必須スキル**:
  - デジタル領域での政策立案または新規サービス開発経験
  - ITサービス・デジタルプロダクトのマネジメント経験5年以上
  - 管理職としての人材マネジメント経験3年以上
- **ソース**: [VOLVE デジタル庁シニアエキスパート](https://volve.co.jp/archives/jobinfo/digital90)（現在は404）

### 非公式の評価軸（採用された人の共通点）

1. **ミッション共鳴**: 「国を変えたい」という意思とスキルの両立が最重要とされている
2. **即戦力性**: ジョブ型採用のため、特定スキルの保有が前提（年齢・経歴不問）
3. **官民折衝力**: スタートアップ的課題解決能力 × 省庁間調整力の両立
4. **高い倫理観**: 国家公務員として求められる倫理水準

### 年収・給与体系

| 区分 | 年収レンジ |
|------|----------|
| 全体平均（OpenWork） | 697万円 |
| 民間専門人材（典型的報告値） | 950万円〜1,000万円（基本給840〜960万円） |
| シニアクラス報告値 | 1,000万〜1,500万円 |

| 当初の目標設計 | 年収換算最大1,100〜1,400万円程度 |

**給与制度の特徴**:
- 民間専門人材（非常勤）は日額給与×勤務日数の計算方式
- アソシエイト職は残業代あり、シニアエキスパート職は俸給表ベースで残業代なし
- 賞与・残業代の有無がアソシエイト職とエキスパート職で異なる

## 論点3: 採用されている人の「スタートアップ・起業家」比率——AIスタートアップ経営者・売却経験者は採用されているか

### 確認されたスタートアップ・起業家経験者の事例

| 人物 | スタートアップ経験 | デジタル庁での関与形態 |
|------|-----------------|---------------------|
| 樫田光 | GrowthCamp共同創業（2020年） | 民間専門人材として正式入庁（2022年〜）、Chief Analytics Officer |
| 石黒卓弥（LayerX執行役員） | 非創業者だがスタートアップ幹部 | 設置準備委員として関与 |
| 安野貴博 | BEDORE創業・MNTSQ共同創業 | ワーキンググループ構成員（外部委員）として関与 |

### 採用傾向の定性評価

**確認されたこと**:
- スタートアップ創業経験者が「正式な民間専門人材（職員）」として採用される事例は存在する（樫田氏）
- ただし「AIスタートアップをExit（売却・上場）した後にデジタル庁職員になった」という事例は今回のリサーチでは確認できなかった
- 「外部委員・ワーキンググループ構成員」という形での関与は、スタートアップ創業者・経営者にも開かれている（安野貴博が典型例）

**民間からの採用で発表された第一弾（2021年）の特徴**:
- 「スタートアップ企業に勤めたり、会社を経営するなどしている20代から60代」が競争率40倍超を勝ち抜いて採用されたと報道されており、起業家・スタートアップ関係者の採用実績は明示されている
- ソース: [Yahoo!ニュース/日テレ](https://news.yahoo.co.jp/articles/b609f00521038f9ae016c615d7d4d88dd0faa37b)

### エンジニア・テクニカル系 vs 経営者・ビジネス系の比率

- **公開情報から推定**: 2021年発足時の採用職種約23職種のうち、技術職（エンジニア・アーキテクト・セキュリティ系）が大多数を占め、PM/マネジメント職が次点、非技術職は少数
- **職種別採用規模（2021年第一回）**: PjM 35名、PM 40名、セキュリティ 43名が目立つ
- **評価**: テクニカル系が人数ベースでは多数派だが、PM・マネジメント系の採用枠も相当数確保されており、二分法的にテクニカル系が圧倒的多数とは言えない
- **スタートアップ経営経験者**: 採用されているが比率の定量データは非公開

## 論点4: デジタル庁外部人材経由で政界・政治活動に関与した前例——「政治キャリアの踏み台」として機能した事例はあるか

### 安野貴博（あんの たかひろ）の事例——最も近い前例

**経歴の流れ（民→関与→政界）**:
1. BEDORE創業（AIチャットボット、後にPKSHA Communicationに吸収）
2. MNTSQ共同創業（リーガルテック、2018年）
3. **デジタル庁「デジタル関係制度改革検討会 デジタル法制ワーキンググループ」構成員（2022年〜）** ← デジタル庁との関与
4. GovTech東京アドバイザー就任（2024年11月）
5. 東京都知事選挙に立候補（2024年）、15万票獲得・5位
6. チームみらい設立（2025年5月）
7. **参議院議員当選（2025年7月）**——比例代表、23万票

**評価**:
- デジタル庁「職員・外部専門人材」ではなく「外部委員・ワーキンググループ構成員」という形での関与であり、「デジタル庁職員が政界入り」とは言えない
- ただし、デジタル庁関連のワーキンググループ関与 → 政界入りのパスが存在することを示す現実の事例として機能している
- **ソース**: [安野貴博Wikipedia](https://ja.wikipedia.org/wiki/%E5%AE%89%E9%87%8E%E8%B2%B4%E5%8D%9A)、[参議院議員プロフィール](https://www.sangiin.go.jp/japanese/joho1/kousei/giin/profile/7025005.htm)

### 「民→官→政界」の他事例

**デジタル大臣歴代の確認**:
- **平井卓也**（初代デジタル大臣）: 民間（広告会社・地元放送局経営）→ 政界（37歳で国会議員、25年以上のキャリア）→ 初代デジタル大臣。「民→政界→デジタル庁大臣」という順序であり、外部人材経由ではない
- **河野太郎・牧島かれん・平将明**: いずれも政治家として先に議員になった後にデジタル大臣に就任するパターン

**デジタル庁職員（外部人材）→政界転身の事例**:
- **今回のリサーチでは確認できなかった**（情報なし）
- デジタル庁は2021年9月設立の比較的新しい組織であり、民間人材として入庁して退庁後に政界入りするケースが出てくるにはまだ年月が浅い可能性がある

**前身組織（IT総合戦略室・CIO補佐官）からの政界転身**:
- 同様に確認できなかった（情報なし）
- CIO補佐官の砂金信一郎氏などは有名人物だが、政界入りの情報なし

## 論点5: デジタル庁AIの具体的な組織・業務内容——外部人材が実際に何をやるのか

### デジタル庁のAI関連組織構造

```
デジタル庁
└── 戦略・組織グループ（G）
    └── AI実装総括班（通称「AI班」）
        ├── ガバメントAI推進
        │   └── 生成AI利用環境「源内（げんない）」の開発・運用
        └── AI実装戦略総括
            └── 省庁横断のAI普及・実装支援
```

**先進的AI利活用アドバイザリーボード（外部諮問機関）**:
- 設置: 2025年6月12日（デジタル社会推進会議議長決定）
- 役割: 政府のAI活用について高度な助言・支援を行う外部専門家組織
- 会合: 第1回2025年9月18日、第2回2026年1月13日、第3回2026年3月10日
- メンバー（確認済み一部）: Kadobayashi（座長）、Nabatame、Yoshinaga、Torisawa、SHIBAYAMA、K. Okada、Kitamura、Yuasa、NAGANUMA（詳細な所属情報はPDFにあるが未取得）
- **ソース**: [デジタル庁 Advanced AI Utilization Advisory Board](https://www.digital.go.jp/en/councils/ai-advisory-board)

### ガバメントAI「源内」の概要

| 項目 | 内容 |
|------|------|
| 開始 | 2025年5月（内製開発） |
| 対象 | デジタル庁全職員（544名が2025年8月時点で利用） |
| 機能 | テキスト生成AI、行政特化AIアプリ20種以上 |
| 外部連携 | OpenAI社とLLM組み込みで協業（2025年〜） |
| 展開計画 | 10万人以上の政府職員への展開を検討中 |
- **ソース**: [デジタル庁ニュース ガバメントAI解説](https://digital-agency-news.digital.go.jp/articles/2025-12-11)、[OpenAI公式発表](https://openai.com/global-affairs/strategic-collaboration-with-japan-digital-agency/)

### 外部人材のタイトルと業務内容

| タイトル | 概要 |
|---------|------|
| シニアエキスパート | 特定領域の最上位専門家。ユニット長を兼務するケースあり。政策判断・戦略立案レベルに関与 |
| エキスパート | 中堅専門職。具体的な施策の設計・実施を担当 |
| シニアアソシエイト | 実務専門職。週2〜3日勤務の非常勤でもなれる職位 |
| アソシエイト | 実務担当。残業代が支給される職位 |
| デジタル制度エキスパート | 法制度のデジタル化専門職 |
| データサイエンティスト | AI・データ分析担当。生成AIの活用実証が主業務 |
| Chief Analytics Officer | データ分析統括（樫田光氏が就任） |

### 政策立案への実際の関与度

**肯定的評価（実態）**:
- シニアエキスパートはプロジェクトの戦略策定から実装まで一貫して関与し、大臣・副大臣との距離も近い（組織図上もシニアエキスパートは幹部に近い位置に記載）
- 樫田光氏のように Chief Analytics Officer として組織横断的なデータ戦略を主導できる
- AI班は「政府による活用をAIの社会実装の起点とする」という国家戦略レベルの目標を掲げており、外部人材が政策の核心部分を担う設計

**課題・制約（OpenWorkクチコミ等より）**:
- 「行政官については基本、他の役所同様に年功序列」という面も残る
- オファーレターの金額が上限で残業代なし等、待遇面の不満もある
- 配属チームによって裁量度に大きなばらつきがある

## 総括メモ

### デジタル庁外部人材のキャリアパスとして西山氏への示唆

1. **AIスタートアップ創業者・経営者は採用される**: 樫田氏（GrowthCamp共同創業→入庁）が直接的な前例
2. **「外部委員→政界」パスの実例あり**: 安野貴博が実証（デジタル法制WGメンバー→参議院議員）
3. **デジタル庁「職員」からの政界転身**: 確認できた事例はなし（組織が若く前例形成中）
4. **関与形態の選択肢**:
   - A) 民間専門人材（正式入庁、常勤・非常勤）: 実務に深く関与、政策影響力大、ただし公務員制約あり
   - B) アドバイザリーボード・外部委員: 政治活動・事業との兼業が容易、ただし関与度は浅い
5. **AI班（源内プロジェクト）**: 外部人材の活躍フィールドとして最も成長中の領域。国家AIインフラを内製するという政治的意義の高いプロジェクト
