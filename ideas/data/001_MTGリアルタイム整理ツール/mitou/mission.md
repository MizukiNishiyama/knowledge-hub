# 会議DX ミッション

## テーマ

**会議のDX（デジタルトランスフォーメーション）** — 会議中のリアルタイムAI支援による会議品質の向上

---

## 背景・課題認識

### 会議の本質的価値

会議とは、会社内における**意思決定の場**である。会議の質向上や効率化は、会社の戦略や経営に直接関わる部分の質向上につながり、不可欠な取り組みである。

### 会議の2類型と属人化の問題

会議には大きく分けて以下の二通りがある。

1. **社外の会議（商談・営業等）**
   - ヒアリングや提案など、営業スキルが**属人化**している
   - トップセールスマンと一般のセールスマンで売上の差が**数十倍**開くことも珍しくない

2. **社内の会議**
   - 5人程度のチーム内会議であっても、メンバーの時給が4,000円であれば、1時間の会議だけで**数万円の価値**がある
   - その質が低ければ、直接的なコスト損失となる

### AIエージェント時代における会議のボトルネック

- 昨今、AIエージェントやAIへの投資が加速している
- AIネイティブな世界では、MCP等を通じて各データベースが接続され、最適なデータのやりとりが行える状況が見えてきている
- そのなかで**会議がボトルネック**になる可能性が高い
  - 意思決定権は当面、人間が持つことが想定される
  - 会議で扱われる事項の方向性決定をAIに委ねることは、現状では文化的に浸透していない

### 既存ソリューションの限界

- AI議事録ツールなどが登場しているが、これは**ミーティング後のまとめ**にとどまっている
- 会議の質向上や効率化に活用されているケースは、大企業を含めて**ほとんどない**

## 我々の提案

### コンセプト

**会議中に会議の質を高めるOSSツール**

### 提供する価値

会議のリアルタイム支援として、以下の機能を通じて会議の高品質化を実現する。

| 機能 | 説明 |
|------|------|
| **リアルタイム要約** | 会議の流れをリアルタイムでまとめる |
| **アジェンダ管理** | アジェンダの進行を管理し、未回収の論点を可視化する |
| **未回収論点の掘り起こし** | 回収されていない議題や質問を検出・提示する |
| **ミスリーディングの指摘** | 話の中で誤解を招く主張や論理の飛躍を指摘する |

### 差別化ポイント

- 既存ツール: 会議**後**のまとめ（事後的・受動的）
- 本ツール: 会議**中**のリアルタイム支援（即時的・能動的）
