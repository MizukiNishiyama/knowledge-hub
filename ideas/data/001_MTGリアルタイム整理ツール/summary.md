# サマリー

updated_at: 2026-03-05

## アイディア概要（1-2文）

会議中の発言をAIがリアルタイムで構造化し、論点の進捗・脱線・未決事項を可視化するB向けSaaS。議事録AIがコモディティ化した今、「会議中のリアルタイム構造化」は明確な空白地帯。ただし核機能の「発言評価（赤ペン）」は導入障壁が高く、Phase分けが必要。

## 現状の評価

- 魅力度: ★★★★☆（AI会議アシスタント市場はCAGR 7-12%で成長中。議事録AIはコモディティ化済みで、「リアルタイム構造化」は明確な次の戦場。ビッグテックも未搭載。ただし時間的猶予は1-2年）
- 実現可能性: ★★★★☆（STT API + LLMストリーミング + Recall.aiで技術的には構築可能。AI Meeting Boardがプロトタイプとして実現可能性を証明済み。課題はリアルタイム遅延の最適化とLLM構造化精度）
- 差別化: ★★★☆☆（「リアルタイム論点構造化」は空白地帯だが、技術的モートは弱い。UX・データ・ネットワーク効果でモートを築く時間との勝負。Fellow.app / SuperInternが最も近い競合）
