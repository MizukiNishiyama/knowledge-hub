# Nujabes-Inspired Sampling Reference

## コンセプト

Nujabes（瀬葉淳）のサンプリング美学を基盤に、既存のドラム・コーラス素材から新しいミュージックを生み出す実験。

Nujabesの音楽的核心は**ループの美学**にある。ジャズ、ソウル、クラシックの断片を切り取り、絶妙な長さのループとして再構築することで、原曲とは全く異なる感情的風景を生み出す。これは単なるコピーではなく、素材の中に眠る「もうひとつの音楽」を発掘する行為である。

## サンプリング手法

本プロジェクトでは以下の手法を採用した。

### 基本手法

| 手法 | 説明 | Nujabesでの使用例 |
|------|------|-------------------|
| **切り取り（Chopping）** | 原曲から特定区間を正確に切り出す | ジャズピアノのフレーズ、弦楽器のパッセージ |
| **テンポ修正（Time Stretching）** | ピッチを保ったままテンポを変更する | 原曲を減速させてレイドバックな雰囲気を生成 |
| **ピッチ修正（Pitch Shifting）** | テンポを保ったまま音程を変更する | サンプルのキーを楽曲に合わせる |
| **逆再生（Reverse）** | オーディオを時間軸反転させる | トランジション、アンビエント要素の生成 |
| **レイヤリング（Layering）** | 複数のサンプルを重ね合わせる | ドラムブレイク＋メロディサンプルの融合 |
| **ローファイ処理（Lo-fi Processing）** | フィルタ・EQで質感を加工する | ダスティでウォームな音像の構築 |

### 複合手法

- **Speed変更**: テンポとピッチを同時に変化させる（レコードの回転数を変えるような効果）
- **EQ + Reverse**: 逆再生にフィルタを組み合わせてドリーミーな質感を生成
- **Chop + Layer**: 異なるソースからの切り出しを重ねて新しい楽曲構造を構築

## 制作物一覧

### Sample 01: Mellow Chorus Loop
- **素材**: コーラス1（30〜38秒）
- **手法**: 切り取り → テンポ92%減速（ピッチ保持）→ 3ループ
- **意図**: Nujabesの楽曲に通底するメロウでレイドバックなグルーヴ感を再現。テンポをわずかに落とすことで、原曲にはなかった「浮遊感」を付与する。
- **出力**: `output/sample_01_mellow_chorus_loop.wav`（約26秒）

### Sample 02: Lo-fi Drum Pattern
- **素材**: ドラム1（60〜64秒）
- **手法**: 切り取り → ローパスフィルタ（8kHz）+ ベースブースト + トレブルカット → 3ループ
- **意図**: Nujabesのトラックにおけるダスティで温かみのあるドラムサウンドを再現。高域を抑え低域を持ち上げることで、ヴィンテージレコードから掘り出したような質感を生成。
- **出力**: `output/sample_02_lofi_drum_pattern.wav`（約12秒）

### Sample 03: Reversed Chorus Dream
- **素材**: コーラス2（50〜58秒）
- **手法**: 切り取り → 逆再生 → ピッチ2半音下げ → 3ループ
- **意図**: 逆再生によって原曲の意味を解体し、ピッチダウンで深みを加える。Nujabesの楽曲に時折現れるアンビエント的・幻想的なパッセージを、サンプリングで再構築する試み。
- **出力**: `output/sample_03_reversed_chorus_dream.wav`（約24秒）

### Sample 04: Layered Jazz Beat
- **素材**: ドラム2（20〜26秒）+ コーラス1（80〜86秒）
- **手法**: 各素材を切り取り → コーラスをピッチ1半音下げ＆音量-6dB → 2素材をミックス → 3ループ
- **意図**: Nujabesの最も象徴的な手法——ドラムブレイクとメロディサンプルのレイヤリング——を再現。ドラムを主軸に、コーラスをバッキングとして薄く重ねることで、一つの「ビート」としての完成度を追求。
- **出力**: `output/sample_04_layered_jazz_beat.wav`（約18秒）

### Sample 05: Chopped & Pitched Soul
- **素材**: コーラス2（100〜110秒）
- **手法**: 切り取り → スピード85%（テンポ＆ピッチ同時低下）→ ローパスフィルタ（10kHz）+ ベースブースト → 3ループ
- **意図**: レコードの回転数を落とすような「チョップド・アンド・スクリュード」的手法。Nujabesがしばしば行った、ソウルやR&Bサンプルを大胆にスロウダウンさせて新たな感情を引き出す技法へのオマージュ。速度低下に伴うピッチの自然な降下が、独特の哀愁と深みを生み出す。
- **出力**: `output/sample_05_chopped_pitched_soul.wav`（約35秒）

## Nujabesのサンプリング哲学

> 「音楽は感情の言語だ。サンプリングとは、他者の感情を借りて、自分だけの物語を紡ぐこと。」

Nujabesのサンプリングには3つの原則がある。

1. **ループの完結性**: 切り取った断片が、それ自体で一つの完結した音楽的世界を形成すること
2. **感情の変換**: 原曲の文脈から切り離された素材に、新しい感情的意味を与えること
3. **空間の創出**: サンプル間の「隙間」を大切にし、音と沈黙のバランスで空間を設計すること

これらの原則を意識しながら、今回の5つのサンプルでは、それぞれ異なるサンプリング手法を通じてNujabes的な音楽空間の構築を試みた。
