## 3. 主力サービス詳細

### 3-1. Hapitas（ハピタス）

#### 基本情報

| 指標 | 数値 |
|---|---|
| 会員数 | 570万人超（2026年時点。540万人突破は2025年3月） |
| 年間流通総額（GMV） | 1,790億円超 |
| 掲載企業数 | 3,800件超 |
| 掲載広告数 | 3,600件以上 |
| 月間PV | 約3,000万PV |

#### サービス概要

ハピタスは「ポイ活（ポイント活動）」のためのポイント還元サイト。ユーザーがハピタスを経由してネットショッピングやサービス申し込みを行うと、通常のポイント（楽天ポイント等）に加えてハピタスポイントが貯まる仕組み。

#### ビジネスモデル（収益構造）

```
[広告主（EC事業者等）]
    |
    | 広告掲載料・成果報酬を支払い
    v
[ハピタス（オズビジョン）] ← ここで一部をマージンとして取得
    |
    | 報酬の一部をポイントとしてユーザーに還元
    v
[ユーザー]
    |
    | ハピタス経由で商品購入・サービス利用
    v
[広告主（EC事業者等）]
```

**具体例**: クレジットカード会社が1件あたり1万円の成果報酬を設定した場合、ハピタスはそのうち約7,000円をユーザーにポイント還元し、残りの約3,000円を自社利益とする。

#### 主な利用カテゴリ

- クレジットカード発行（高額案件が多い）
- ネットショッピング（楽天市場、Yahoo!ショッピング等）
- 証券口座・銀行口座開設
- 旅行予約
- アプリダウンロード
- 各種サービス無料登録

#### ポイント交換先

ハピタスポイント（1pt = 1円）は以下に交換可能:

- 現金（各種銀行振込）
- 電子マネー（PayPay、楽天ポイント等）
- ギフト券（Amazonギフト券等）
- マイル（ANAマイル、JALマイル等）
- 他社ポイント

**特徴**: ポイント交換手数料が全て無料

### 3-2. Pollet（ポレット）

不要品を手軽にキャッシュ化できる買取サービス。

- **対象**: 使わなくなった「モノ」、余った「外貨」、「商品券」など
- **手順**: 「まとめて、詰めて、送るだけ」の3ステップ
- **特徴**: スマホアプリで完結、査定後にVisaプリペイドカードにチャージ

もともとPollet株式会社として分社化されていたが、その後オズビジョンに吸収合併されている。

### 3-3. OZASP（オザップ）

完全招待制のクローズドASP（アフィリエイト・サービス・プロバイダー）。

- 自社運営のハピタスをはじめ、インセンティブメディア・インフルエンサー・記事メディアなど多様なパートナーと連携
- 量ではなく「質」にこだわった成果を提供する広告プラットフォーム
- 広告主に対してハピタスの570万人ユーザー基盤を活用したマーケティング支援を行う

## 6. 競合他社との比較・差別化

### 主要競合

| サイト名 | 運営会社 | 会員数 | 特徴 |
|---|---|---|---|
| **モッピー** | セレス（東証プライム上場） | 約900万人 | 業界最大の会員数。広告掲載数5,000件以上。ゲームコンテンツが充実 |
| **ポイントインカム** | ファイブゲート | 約500万人 | 独自のステータスボーナスが充実。一度上がったランクが下がらない |
| **ちょびリッチ** | ちょびリッチ | 約400万人 | ちょびガチャなど無料コンテンツが豊富 |
| **ポイントタウン** | GMOメディア | 約830万人 | GMOグループの信頼性。ミニゲームが充実 |
| **ハピタス** | オズビジョン | 約570万人 | 高還元率・交換手数料無料。ショッピング系に強い |

### ハピタスの差別化ポイント・強み

1. **ポイント交換手数料が完全無料**: 現金・電子マネー・ギフト券への交換手数料が一切かからない。競合の多くは手数料を徴収する
2. **高還元率**: 特にクレジットカード発行・金融系案件で業界トップクラスの還元率を提供
3. **お買い物あんしん保証**: ハピタス経由で買い物をしてポイントが付与されなかった場合の保証制度がある（ゴールド会員は14日以内に対応）
4. **シンプルなUI**: ゲームや広告クリック系のコンテンツを絞り、ショッピング還元に特化したシンプルな設計
5. **年間流通総額1,790億円**: 国内ポイントサイトでトップクラスのGMV
6. **独自案件**: ハピタスでしか取り扱いがない広告案件の存在

### ハピタスの弱み・デメリット

- モッピーと比較して広告掲載数がやや少ない（3,600件 vs 5,000件）
- 無料ゲームやアンケートなどの「コツコツ系」コンテンツが少ない
- アプリの機能がモッピー等と比較してやや限定的

## 8. ユーザーの継続利用を促す仕組み

### 8-1. 会員ランク制度

ハピタスは4段階の会員ランク制度を導入し、継続利用を促進している。

| ランク | 獲得ポイント条件（過去12ヶ月） | 利用広告件数条件（過去12ヶ月） | ポイント交換ボーナス |
|---|---|---|---|
| ゴールド | 5,000pt以上 | 12件以上 | 最大+5% |
| シルバー | 3,000pt以上 | 8件以上 | 最大+3% |
| ブロンズ | 1,000pt以上 | 4件以上 | 最大+1% |
| 一般 | - | - | なし |

**ルール**:
- 昇格は毎日集計、条件達成の翌日に反映
- 降格は毎月末集計、翌月1日に反映
- 同じ案件を複数回利用しても件数カウントは1件（異なる案件の利用が必要）
- ゴールド会員限定で「お買い物あんしん保証 14日以内対応」の特典あり

### 8-2. ハピタス宝くじ

- 毎月開催される無料抽選イベント
- 宝くじ交換券を毎日コツコツ集められる仕組み
- 当選すると最大30,000ハピタスポイント獲得
- 毎日サイトにアクセスする動機を作るデイリーエンゲージメント施策

### 8-3. スタンプラリー

- 新規登録から7日間限定のスタンプラリーキャンペーン
- スタンプを集めると最大500ptプレゼント
- 初期の利用習慣の定着を促進するオンボーディング施策

### 8-4. 友達紹介プログラム

紹介者と被紹介者の双方にインセンティブを付与する仕組み。

**紹介者への報酬（ダウン報酬）**:

| 累計紹介人数 | 紹介報酬率 |
|---|---|
| 1人 | 10% |
| 5人 | 15% |
| 10人 | 20% |
| 50人 | 25% |
| 100人 | 30% |
| 500人 | 35% |
| 1,000人 | 40% |

- 紹介した友人が獲得したポイントの一定割合が、紹介者に毎月自動付与される
- 紹介人数が増えるほど報酬率が上がるため、ブロガーやインフルエンサーの積極的な紹介を促進

**被紹介者への特典**:
- 友達紹介経由の新規登録で最大2,600ポイント（条件達成が必要）

## 10. まとめ -- オズビジョンの全体評価

### 強み

- **国内トップクラスのGMV（1,790億円）**: 流通総額で業界随一の規模
- **570万人の会員基盤**: 購買意欲の高いユーザー層を保有
- **交換手数料無料**: ユーザーフレンドリーな設計がLTV向上に寄与
- **高還元率**: 特に金融系案件で競合優位
- **ティール組織の実践**: 自律分散型の組織文化が柔軟な事業運営を支える
- **収益基盤の安定性**: 成果報酬型モデルで広告主とユーザー双方にメリットを提供

### 課題・リスク

- **未上場**: モッピー（セレス）やポイントタウン（GMOメディア）のように上場企業が運営する競合と比較すると、資金調達面での制約
- **ポイントサイト市場の競争激化**: 各サイトの還元率競争が続いており、マージンの圧縮リスク
- **ユーザー獲得コスト**: テレビCMやインフルエンサー施策など、マーケティングコストの上昇
- **規制リスク**: ポイント関連の法規制変更による事業影響の可能性

### 今後の展望

- 購買データを活用したリサーチ事業への展開（広告以外の収益多角化）
- リクロマート買収によるリードメディア事業の強化
- Z世代への訴求強化（TikTok等のSNSマーケティング）
- CTO採用によるプロダクト開発体制の刷新
