## なぜ「広告最適化」なのか — 構造的な隙がある

### OOH広告市場の現状

日本のOOH広告市場は約4,487億円（2024年）。そのうちDOOH（デジタルOOH）は約1,110億円（2025年見込み）で、**前年比116%**と急成長している。2030年には1,647億円に達する見通し。

しかし、この市場には**根本的な課題**がある。

### OOH広告の最大の痛み: 「効果が見えない」

デジタル広告（Web/SNS）では当たり前の指標が、OOH広告では存在しない。

| 項目 | デジタル広告 | OOH広告（現状） |
|---|---|---|
| 誰に届いたか | 個人レベルで特定可能 | **不明**（推計ベース） |
| 何人に届いたか | 正確なインプレッション | **不明**（通行量からの推計） |
| 見たかどうか | Viewability基準あり（MRC） | **不明**（視認の確認手段がない） |
| 属性 | 年齢・性別・興味関心 | **不明** |
| 効果測定 | CPC/CPA/CVRでリアルタイム | **不可能**（事後アンケート等） |
| 課金モデル | 成果連動（CPC/CPA） | **期間固定の枠買い** |

広告主の本音は明確だ。**「OOHに出稿したいが、効果が見えないから社内で予算を通せない」**。

### 業界の動き — 今まさにインフラが整いつつある

- **2025年9月**: 日本OOHメジャメント協会（JOAA）設立（電通・博報堂DYMPほか13社参画）
- **2026年3月**: JOAAがOOH広告メジャメントデータの提供を開始予定
- **LIVE BOARD**: NTTドコモ+電通+博報堂DYMPの合弁。全国16,000面以上のDOOHでpDOOH（プログラマティックDOOH）を展開中

つまり、**「OOH広告を効果測定可能にしたい」という業界全体の意志が動き出した**タイミングである。

### しかし、LIVE BOARDの計測には限界がある

LIVE BOARDのインプレッション推計は**NTTドコモの携帯キャリアデータ（位置情報）ベース**。これは統計的な通行量の推計であり、**「実際にサイネージの前にいる人の属性」をリアルタイムで見ているわけではない**。

ここに**AIカメラが埋める隙**がある。

### コア技術: 視線トラッキングによる「真のインプレッション」計測

本サービスの技術的な核は、**デジタルサイネージに設置したカメラで通行者の視線トラッキングを行い、「属性」と「真のインプレッション」を同時に計測する**ことにある。

既存の計測手法との違いを整理する。

| 計測手法 | 何を測っているか | 精度 |
|---|---|---|
| 携帯位置情報（LIVE BOARD） | サイネージ付近にいた人の推計数 | 「近くを通った」だけで見たかは不明 |
| 通行量カウント（既存AIカメラ） | サイネージ前を通過した人数 | 「通過した」だけで視認したかは不明 |
| **視線トラッキング（本サービス）** | **サイネージを実際に見た人の数と属性** | **「見た」ことを視線で確認** |

これは広告業界における**Viewability（視認性）**の概念をOOHに持ち込むものである。デジタル広告では「広告がビューポートに表示された」だけでなく「一定時間以上表示された」ことをもってインプレッションとする基準（MRC基準）が標準化されている。OOH広告にはこの基準が存在しない。

視線トラッキングにより、以下の指標を提供できる。

- **VAC（Visibility Adjusted Contact）**: 通行者のうち、実際にサイネージを視認した人数。WOO（World Out of Home Organization）が推奨するグローバル標準指標
- **属性別インプレッション**: 「30代女性が1日に320回視認」等、属性×視認のクロスデータ
- **平均注視時間**: 広告クリエイティブごとに、どの属性がどれだけの時間視認したか
- **注視率（Attention Rate）**: 通行者のうち何%がサイネージを視認したか。広告枠の価値を定量化する指標

これにより、広告主に対して**「この枠に出稿すれば、30代女性に1日320回"見られる"」**と、推計ではなく実測ベースで提示できる。LIVE BOARDの「近くを通った推計人数」とは質的に異なるデータであり、**CPMプレミアムの根拠**になる。

#### 技術的な実現方法

視線トラッキングは以下のOSS/技術の組み合わせで実現可能。

| 要素 | 技術 | 役割 |
|---|---|---|
| 顔検出 | YOLOv8-face / RetinaFace | サイネージ方向を向いている顔を検出 |
| 視線推定 | MediaPipe Face Mesh / L2CS-Net / GazeML | 顔のランドマークから視線方向を推定 |
| 属性推定 | DeepFace / InsightFace | 年齢・性別を推定 |
| トラッキング | ByteTrack / DeepSORT | 同一人物の追跡（重複カウント防止） |
| エッジ処理 | NVIDIA Jetson Orin Nano | サイネージ側で完結。映像を外部送信しない |

サイネージ上部または近傍にカメラを1台設置し、正面から近づく人物の顔の向き・視線方向を推定する。サイネージ方向への注視を検出した時点で「視認インプレッション」としてカウントし、同時に属性を推定する。映像はエッジデバイス上で処理され、統計データのみがクラウドに送信されるため、プライバシーリスクを最小化できる。

## ビジネスモデルの骨格

### 一言で言うと

**「視線トラッキングによる"真のインプレッション"計測で、OOH広告にデジタル広告並みのターゲティングと効果測定を持ち込むアドテクレイヤー」**

### ポジショニング

```
既存の人流分析ツール       → 施設オーナーに「計測」を売る（SaaS）
LIVE BOARD / pDOOH       → 広告主に「配信面」を売る（CPM）
本サービス（提案）         → 広告主に「属性データ付きの効果測定」を売る
```

既存の人流分析ツール（OPTiM、coumera等）は**施設オーナーがカメラを買って自分で分析する**モデル。LIVE BOARDは**配信プラットフォーム**だが計測は携帯位置情報ベース。

本サービスは、**サイネージにカメラを設置し、視線トラッキングで「実際に見た人」の属性をリアルタイム計測して広告配信に接続する**ことで、両者の間を埋める。

### 課金モデル

**属性ターゲティング付きCPM（Cost Per Mille）課金**

- 通常のDOOH CPM: 数百円〜数千円/1,000imp
- 属性ターゲティング付きCPM: **プレミアム30-50%上乗せ**
- 本サービスの取り分: 広告収益の**20-30%をレベニューシェア**
- 残りはサイネージオーナー / 媒体社に分配

広告主は「30代女性が実際に見た回数」に基づいて課金されるなら、通常のOOHより高いCPMを払う合理性がある。「通行量の推計」ではなく「視認の実測」に基づくインプレッションは、広告主にとってデジタル広告に近い信頼性を持つ。

## なぜ既存プレイヤーがこれをやっていないか

| プレイヤー | やらない理由 |
|---|---|
| LIVE BOARD | ドコモの位置情報データがコア資産。カメラベースに移行するインセンティブがない。むしろ補完的に使いたい側 |
| OPTiM / coumera | ビジネスモデルがSaaS月額。広告のバリューチェーンに入るノウハウ・営業チャネルがない |
| NEC FieldAnalyst | 2007年設計。大企業向け個別案件が主体で、プラットフォーム化する動機が薄い |
| AIインサイト（アビックス） | サイネージ一体型ハードウェア販売が主軸。アドテクプラットフォームではない |
| Quividi（仏） | グローバルNo.1のDOOHオーディエンス計測だが、日本市場への本格参入はまだ |

つまり、**人流分析プレイヤーは「計測SaaS」に閉じており、広告プレイヤーは「カメラベースのリアルタイム属性計測」を持っていない**。この2つをつなぐポジションが空いている。

## Go-to-Market 戦略

### Phase 1: PoC（0-6ヶ月）— 1拠点で実証

**目的**: 「AIカメラ属性データでOOH広告のCPMを上げられる」ことを実証する

- 1つの商業施設 or 駅構内のサイネージに、AIカメラ1-2台を設置
- 1ヶ月間の属性データ（年齢・性別・時間帯別の通行者構成）を収集
- 「このサイネージは平日昼に30代女性が40%を占める」等のレポートを作成
- サイネージオーナーまたは広告代理店にレポートを提示し、**属性付きCPMで広告を販売できるか**を検証

**必要リソース**:
- エッジデバイス（Jetson Orin Nano × 2台 ≒ 10万円）
- 開発工数（YOLOv8 + DeepFace + RTSP入力 + ダッシュボード）
- サイネージ設置場所1箇所の確保（知人経由 or コールドアプローチ）

### Phase 2: 広告代理店との連携（6-12ヶ月）— 収益化の検証

**目的**: 実際に属性ターゲティング付きCPMで広告を販売し、広告主の支払意思を確認する

- Phase 1のデータをもとに、広告代理店（中小のOOH専業が望ましい）にアプローチ
- 「属性データ付きのOOH広告枠」として、通常CPMの30-50%プレミアムで販売可能かテスト
- 同時にサイネージ設置場所を5-10拠点に拡大
- 属性連動の配信切り替え機能（男性が多い時間帯→男性向け広告）をMVP実装

### Phase 3: プラットフォーム化（12-24ヶ月）— スケール

**目的**: 面数を増やし、pDOOHエコシステムとの接続を開始

- LIVE BOARD SSPとの技術連携を模索（属性データをSSPに供給する形）
- サイネージ設置場所を50-100拠点に拡大
- DSP経由でのプログラマティック配信に対応
- ネットワーク効果: 面数が増えるほど広告主にとって魅力的になり、広告主が増えるほどサイネージオーナーの参加動機が高まる

## 競合との差別化 — Quividiとの比較

最大の潜在競合は**Quividi（フランス）**。DOOH向けAIオーディエンス計測のグローバルNo.1で、月間10億以上の顔を検出・分析している。

| 項目 | Quividi | 本サービス（提案） |
|---|---|---|
| 拠点 | フランス（グローバル展開） | 日本特化 |
| 日本市場 | 本格未参入 | ファーストムーバー |
| ビジネスモデル | 計測SaaS（媒体社に販売） | 広告レベニューシェア |
| 言語・サポート | 英語 | 日本語ネイティブ |
| 営業チャネル | 海外大手媒体社経由 | 日本の中小サイネージオーナーに直接 |
| 日本の広告商慣習 | 理解が浅い | 理解して設計可能 |

Quividiが日本に本格参入するまでの**時間的アドバンテージ**と、日本の広告商慣習（代理店構造、純広告文化）に最適化した**ローカライズ**が差別化要素になる。ただし、これは永続的なモートではない。

## リスクと対策

### 1. サイネージ設置場所の獲得（最大のボトルネック）

- **リスク**: カメラを設置させてもらえなければ何も始まらない
- **対策**: 「無料で設置、広告収益を分配」というオーナー側のコスト0モデルで攻める。最初の1拠点は知人経由や小規模施設から

### 2. 広告主の支払意思

- **リスク**: 「属性データが付いても、OOHにそこまで払わない」可能性
- **対策**: Phase 1で実証。最悪、属性ターゲティングCPMが通らなくても、「計測レポート」単体をサイネージオーナーにSaaS販売するフォールバックがある

### 3. プライバシー

- **リスク**: カメラ映像からの属性推定に対する社会的反発、法規制リスク
- **対策**: エッジ処理で映像を即時破棄。個人を特定しない統計データのみを伝送。カメラ設置場所での告知義務を遵守。JOAAの計測ガイドラインに準拠

### 4. Quividiの日本参入

- **リスク**: グローバルNo.1が日本に来たら、技術力・実績で負ける
- **対策**: 先に面数を押さえ、サイネージオーナーとの関係性を構築する。計測SaaSではなく広告レベニューシェアという異なるビジネスモデルで、Quividiの「顧客」ではなく「パートナー」になる道もある

### 5. LIVE BOARDとの関係

- **リスク**: LIVE BOARDがカメラベース計測を自前で始める可能性
- **対策**: LIVE BOARDの補完的ポジション（位置情報ベース計測の精度を上げるカメラデータ提供元）を狙う。敵対ではなくエコシステムに入る戦略

## タイミングが「今」である理由

1. **JOAA設立（2025年9月）**: 業界が「OOH広告の効果測定を標準化しよう」と動き出した。インプレッション計測のニーズが顕在化する
2. **JOAAメジャメントデータ提供開始（2026年3月予定）**: 業界共通指標ができれば、それを高精度に計測できるプレイヤーへの需要が生まれる
3. **pDOOHの普及前夜**: 日本のpDOOHは欧米に大幅に遅れている。これから急速に整備されるタイミングで、カメラベースの属性データ供給元として入り込める
4. **技術の成熟**: YOLOv8+DeepFace+Jetsonで実用的な精度が低コストで実現可能になった。2-3年前には困難だった
