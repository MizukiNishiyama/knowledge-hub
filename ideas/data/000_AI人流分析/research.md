## 株式会社ロフトの現状整理

### 基本情報
- 売上高: 1,071億円（2024年2月期）
- 店舗数: 国内151直営 + 26FC + 海外8店舗（2025年5月末時点）
- アプリ会員: 約900万人（2025年5月時点）
- 親会社: セブン&アイ・ホールディングス → ヨーク・ホールディングス傘下
- 客層: 20〜30代女性中心。トレンド感度の高い雑貨・コスメ・文具の購買層

### 経営環境
- **渋谷ロフト閉店（2025年4月）**: 35年の旗艦店が閉店。大型店舗のコスト構造の限界を象徴
- **梅田ロフト閉店→新生梅田ロフトとして移転再開（2025年5月）**: フロア面積を圧縮し、体験型・デジタル活用店舗へ刷新
- **EC競合**: 雑貨・コスメはEC移行が進み、リアル店舗の「行く理由」の再構築が急務
- **没個性化の指摘**: 東急ハンズとの差別化が曖昧。「どこにでもある雑貨屋」化のリスク

### リテールメディア事業（2025年5月〜）

ロフトはアドインテと提携し、リテールメディア事業を開始済み。

| メニュー | 内容 | 展開規模 |
|---|---|---|
| デジタルサイネージ広告 | 全国店舗にサイネージ設置、ブランド広告を配信 | 132店舗 |
| レジ上サイネージ広告 | 旗艦店のレジ横にサイネージ設置 | 3店舗（梅田・渋谷・銀座） |
| アプリ内ターゲティング広告 | 900万アプリ会員にアプリ内広告配信 | 全会員 |
| サンプリングボックス「ロフボ」 | 自動販売機型の試供品提供。アプリ会員限定 | 3旗艦店 |

**成果**: ロフボ設置店舗でアプリお気に入り登録数が前年比270%。休眠会員の掘り起こしに成功。

**現状の計測手法**: アプリ会員のID-POSデータ + アドインテのAIビーコン（Wi-Fi検知ベースの来店推計）。**AIカメラによるリアルタイム属性分析は未導入**。

## 競合サービス（ロフトに提案しうるAI人流分析・リテールメディア関連）

### アドインテ（AdInte）— ロフトの既存パートナー
- URL: https://adinte.co.jp/
- 概要: AIビーコン + CDPベースのリテールメディアプラットフォーム。ID-POS連携で広告効果を計測
- ターゲット: 流通小売チェーン（ロフト、その他大手小売）
- 価格帯: 非公開（業務委託契約ベース）
- 強み: ロフトとの既存関係。ID-POSデータ4万店・流通総額13兆円のネットワーク。博報堂との連携
- 弱み: AIビーコンはWi-Fi検知ベースで属性推定精度が低い。カメラベースのリアルタイム属性分析機能は不明
- 参考になる点: **ロフトに新規提案する場合、アドインテは最大の障壁であり、同時に最大の協業先候補**

### グローリー AIカメラ分析サービス
- URL: https://www.glory.co.jp/company/news/detail/id=3070
- 概要: 顔の特徴量 + POSデータでデジタルサイネージ広告の購買効果を可視化。業界初の顔認証×購買連携
- ターゲット: スーパー、コンビニ、ドラッグストア
- 価格帯: 非公開
- 強み: 顔特徴量と購買データの紐付けによる広告効果の因果関係測定。業界初のポジション
- 弱み: 2025年9月開始で実績蓄積中。雑貨小売への適用実績は未確認
- 参考になる点: **ロフトの文脈では「サイネージ広告を見た人が実際に買ったか」を計測できる点が刺さる可能性大**

### OPTiM AI Camera
- URL: https://www.optim.co.jp/optim-ai-camera/
- 概要: クラウドAI画像解析。混雑可視化・人数カウント・属性分析など300種類以上の解析
- ターゲット: 小売店舗、鉄道・交通機関、医療機関
- 価格帯: 月額2,980円/1カメラ〜
- 強み: 低価格。映像即時破棄でプライバシー配慮
- 弱み: 汎用ツールのため、リテールメディア文脈での広告効果測定には追加開発が必要
- 参考になる点: 月額2,980円の価格設定。ロフト132店舗に導入した場合の月額コスト感（約40万円/月〜）

### Vieureka（パナソニック子会社）
- URL: https://www.vieureka.com/analysis/
- 概要: パナソニック製AIカメラで来店客の人数・属性（性別・年齢・滞留時間）をデータ化
- ターゲット: 小売チェーン、商業施設（サツドラ等に実績）
- 価格帯: AIカメラ+Webアプリセットで約50万円〜
- 強み: パナソニックのブランド力。エッジAI処理。マスク装着でも属性推定可能
- 弱み: 初期導入コストが高い。132店舗×50万円=6,600万円の初期投資は障壁
- 参考になる点: セブン&アイグループ内でパナソニック製品が採用される可能性。グループ調達の力学

### Quividi（フランス）
- URL: https://quividi.com/
- 概要: DOOH向けオーディエンス計測のグローバルNo.1。月間10億以上の顔を検出・分析。80カ国以上で展開
- ターゲット: DOOHネットワーク、リテールメディア
- 価格帯: 非公開
- 強み: 世界最大の実績。Hikvisionカメラとの統合。リテールメディア向け機能を拡充中
- 弱み: 日本市場での知名度が低い。DGCTがアジア代理店だが日本語サポートの深度は不明
- 参考になる点: **リテールメディア×AIカメラ属性分析のグローバルスタンダード。ロフトが本格導入を検討するなら比較対象になる**

### NEC FieldAnalyst
- URL: https://www.nec-solutioninnovators.co.jp/sl/fieldanalyst/
- 概要: サイネージ前の通過・視聴人数、性別/年齢/マスク装着有無、視聴時間を測定
- ターゲット: 大型商業施設、デジタルサイネージ事業者
- 価格帯: 非公開（大企業向け個別見積もり）
- 強み: NECの顔認識技術（世界トップクラス）。「場の価値」の数値化
- 弱み: 2007年設計。個別見積もりで中小にはハードルが高い
- 参考になる点: サイネージ連携の先駆者。視聴時間・距離の計測指標

### coumera（GRoooVE）
- URL: https://coumera.com/
- 概要: 低価格の店舗計測ツール。入店カウント精度95%以上
- ターゲット: D2C、小売店、ポップアップストア（800店舗超の実績）
- 価格帯: 月額18,000円（税込）〜
- 強み: 業界最安値クラス。設置工事不要。最短1カ月利用可
- 弱み: 基本計測特化。属性分析やサイネージ連携は限定的
- 参考になる点: 低価格・短期導入モデルのリファレンス

### TAM / SAM / SOM（ロフト文脈）

- **TAM**: 日本のリテールメディア + 店舗内デジタルサイネージ市場 ≒ 約1,500〜2,000億円
- **SAM**: 雑貨・バラエティショップチェーン向けAIカメラ人流分析 + サイネージ連携 ≒ 推定30〜80億円
  - ロフト（132店舗）+ 無印良品（500+店舗）+ ドンキ（700+店舗）+ 東急ハンズ/ハンズ + その他
- **SOM**: ロフト132店舗へのAIカメラ導入 + 運用SaaS ≒ 推定1〜3億円/年
  - 132店舗 × 月額5〜20万円 = 年間7,920万〜3.2億円

### 関連トレンド

1. **リテールメディア急拡大**: 小売企業が自社の顧客接点（店舗・アプリ・EC）を「広告媒体」として収益化する動きが加速。米国ではAmazon広告がGoogle/Metaに次ぐ第3極に
2. **サイネージ×AIカメラの融合**: グローリーの業界初サービス（2025年9月〜）に象徴されるように、「サイネージを見た人が買ったか」の因果を測る技術が実用段階に
3. **JOAA設立（2025年9月）**: OOH広告の効果測定標準化が業界コンセンサスに。2026年3月にメジャメントデータ提供開始予定
4. **プライバシーファースト設計**: エッジAI処理で映像を即時破棄し、統計データのみ伝送する設計が標準化

## 技術的な実現手段

### ロフト店舗へのAIカメラ導入アーキテクチャ（想定）

```
[ロフト店舗]
  ├─ 既設防犯カメラ（RTSP）or 新設AIカメラ
  ├─ エッジデバイス（Jetson Orin Nano / Raspberry Pi 5 + Hailo-8L）
  │   ├─ YOLOv8: 人物検出
  │   ├─ DeepFace / InsightFace: 年齢・性別推定
  │   ├─ ByteTrack: 同一人物追跡（重複カウント防止）
  │   └─ MediaPipe Face Mesh: 視線方向推定（サイネージ向き）
  ├─ 統計データのみクラウドに送信（映像はエッジで即時破棄）
  └─ ダッシュボード / API

[クラウド]
  ├─ 属性別来店データ集計
  ├─ 時間帯・曜日・季節別トレンド
  ├─ サイネージ視認率レポート
  └─ アドインテCDP / ロフトID-POSとのデータ連携
```

### ロフト店舗特有の技術的課題

| 課題 | 詳細 | 対策案 |
|---|---|---|
| 多層フロア | ロフト大型店はB1〜6Fなど複数階構成。フロア間の人流追跡が困難 | フロアごとに独立計測し、アプリ会員はBLE/Wi-Fiで紐付け |
| 什器の高さ | 棚が高く、カメラの画角が遮られやすい | 天井カメラ + 通路端のカメラ配置。フィッシュアイレンズ活用 |
| 照明のばらつき | 催事スペースとレジ周りで照度が大きく異なる | モデルのfine-tuning。低照度対応カメラの選定 |
| ネットワーク | 店舗Wi-Fiの帯域がエッジデバイスのデータ送信に十分か | エッジで完結し、統計データのみ数KB/分で送信。帯域負荷は軽微 |
