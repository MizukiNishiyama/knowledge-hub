# サマリー

updated_at: 2026-02-23

## アイディア概要

カメラ映像をAIでリアルタイム解析し、来訪者の属性（年齢層・性別・グループ構成）と行動パターンを即時に可視化するB向けサービス。商業施設向け人流分析とデジタルサイネージ連携の2軸で展開を想定。

## 現状の評価

- 魅力度: ★★★★☆（グローバルCAGR 15〜23%の高成長市場。日本のリテールテック市場も2030年に5,553億円へ拡大見込み）
- 実現可能性: ★★★★☆（YOLOv8+DeepSORT+Jetsonなど、OSSベースで技術的には構築可能。ただしプロダクト品質にするには相応の開発工数が必要）
- 差別化: ★★☆☆☆（競合多数。OPTiM・Vieureka・coumera・NEC等が先行。OSSベースで参入障壁も低い。明確な差別化軸の確立が最大の課題）

## 未解決の論点

1. **ターゲットの絞り込み**: 商業施設 / 小売店 / サイネージ事業者のどこから攻めるか未定
2. **差別化戦略**: 低価格路線（coumera対抗）/ 高機能路線（NEC対抗）/ サイネージ特化 / 既存カメラ活用のゼロハードウェアコスト、のいずれで戦うか
3. **プライバシー・法務リスク**: カメラ映像からの属性推定に関する法的整理がまだされていない
4. **ビジネスモデル**: SaaS月額 / 広告レベニューシェア / ハードウェア込みなど、課金モデルが未確定
5. **MVP定義**: 最小スコープ（1カメラ×人数カウント+属性推定？）と開発コストの見積もりが必要
6. **大手との競争**: NEC・パナソニックが本格参入済み。スタートアップとしてどう戦うかの戦略が未整理
