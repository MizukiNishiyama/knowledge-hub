## 競合サービス

### Rippling Talent Signal（最重要競合）
- URL: https://www.rippling.com/talent-signal
- 概要: AIが従業員の実際の業務データ（GitHub / Salesforce / Gong / Zendesk）を読み取り、入社90日時点で「High Potential / Typical / Pay Attention」の3段階評価を自動生成するツール
- ターゲット: エンジニア・営業・カスタマーサポート職のパフォーマンス可視化（主に米国企業）
- 価格帯: Rippling HRプラットフォームの一部として提供（HRプラットフォーム全体で$8〜/PEPM）
- 強み: **memo.mdの構想に最も近い**。CRM・コードリポジトリ等の実業務データを直接分析し、具体的なエビデンス付きで評価を提示。OpenAI / Anthropicのモデルを活用。自社従業員+オプトイン顧客のデータで学習済み
- 弱み: 入社90日の新人評価に特化しており、既存社員の継続評価や人事評価サイクル全体をカバーしていない。日本未展開。対象職種が限定的（エンジニア・営業・CS）
- 参考になる点: 「AIが評価する」のではなく「AIがシグナルを出し、マネージャーが判断する」設計。thumbs up/downでマネージャーがフィードバックを返す仕組みでAI単独評価への抵抗感を緩和

### Lattice
- URL: https://lattice.com
- 概要: パフォーマンスマネジメント・OKR・エンゲージメント・報酬管理を統合したHRプラットフォーム。AI機能でレビューの文章生成・キャリブレーション支援を提供
- ターゲット: 100〜5,000人規模の米国企業（IT・テック企業中心）
- 価格帯: $11/PEPM（Talent Management）〜 $22/PEPM（全機能）
- 強み: Slack連携で日常的なフィードバックを収集。AI Calibrationでマネージャー間の評価バラつきを自動検知。エンゲージメントサーベイとの統合で離職予測も可能
- 弱み: AIは「レビュー文章の下書き支援」が中心で、業務データからの自動評価は行わない。日本語対応なし
- 参考になる点: 2023年にAI機能（Reviews with AI）を発表したが、「AIが従業員をスコアリングする」という方針に対してSNSで強い反発を受け、方針を「AIはマネージャーの支援ツール」にピボットした教訓

### Effy AI
- URL: https://www.effy.ai
- 概要: AI-first の360度フィードバックプラットフォーム。レビューフォームの自動生成、フィードバックの要約、バイアス検知を提供
- ターゲット: 50〜500人規模の企業。Slack / Teams上で直接レビューを実施可能
- 価格帯: Free（無制限メンバー・レビュー） / Pro $75/スペース/月（AIレポート30件付き、追加$24/件）
- 強み: 無料プランで無制限利用可能な圧倒的な価格競争力。AIがフィードバックのトーンを調整し、バイアス（性別・年齢等の不公平な表現）を自動検知。管理工数を70%削減する実績
- 弱み: 360度フィードバックに特化しており、業務データ（Slack発言・CRM実績等）からの自動評価は行わない。人間がフィードバックを書く前提
- 参考になる点: 「バイアス検知」はAI人事評価における最大の懸念の一つであり、この機能を前面に出すポジショニングは参考になる

### HRBrain
- URL: https://www.hrbrain.jp
- 概要: 人事評価クラウドからスタートし、タレントマネジメント・組織診断サーベイ等に拡張した日本のHRテック企業
- ターゲット: 100〜数千人規模の日本企業（全業種）
- 価格帯: 約70,000円/月（100名規模）。従業員規模に応じた見積もり制
- 強み: 累計導入3,500社。目標管理（MBO / OKR）、評価ワークフロー、1on1記録、組織サーベイを一体提供。日本の人事慣行に最適化
- 弱み: AI機能は限定的（レポート生成程度）。業務データからの自動評価やLLMによる分析は提供していない。既存の評価プロセスのデジタル化が主軸で、評価自体の自動化は行わない
- 参考になる点: 日本市場での価格感・機能期待値のベンチマーク。「人事評価クラウド」から周辺機能に拡張した成長パターン

### カオナビ
- URL: https://www.kaonavi.jp
- 概要: 日本のタレントマネジメントシステムのシェアNo.1（9.9%）。人材情報の一元管理・可視化が強み
- ターゲット: 100〜数万人規模の日本企業（全業種、エンタープライズに強い）
- 価格帯: 非公開（見積もり制。推定: 100名規模で月額5-10万円）
- 強み: 市場シェア1位。人材データベース・評価ワークフロー・配置シミュレーション・スキル管理を統合。大企業の人事部門に深く浸透
- 弱み: AIによる評価自動化は未実装。「人材情報の可視化」が主軸で、評価の質向上よりも管理効率化にフォーカス
- 参考になる点: 日本の人事評価ツール市場のトッププレイヤー。カオナビですらAI自動評価に踏み込んでいない（＝市場にニーズがないのか、リスクが大きすぎるのか、判断が分かれる）

### Insight Edge（住友商事グループの実証実験）
- URL: 非公開（住友商事グループ内の実験）
- 概要: Slack・Google Drive・GitHubの活動データを収集し、AIで社員の貢献度を分析する実証実験を実施
- ターゲット: 住友商事グループ内（社内ツール）
- 価格帯: 社内プロジェクトのため外販なし
- 強み: memo.mdの構想に近い「複数チャネルのデータ統合×AI分析」を実際に試みている
- 弱み: 実証段階であり、プロダクト化されていない。結果の詳細は非公開
- 参考になる点: 大企業が「やってみた」が、プロダクト化に至っていない事実は重要。技術的な問題か、組織的な抵抗か、あるいはROIが出なかったか

- 概要: AIを活用した人事評価クラウド。目標設定の添削・評価コメントの自動生成機能を搭載
- ターゲット: 中小〜中堅企業（日本）
- 価格帯: 非公開（見積もり制）
- 強み: 「AIが評価を支援する」をいち早く打ち出した日本プレイヤー。人事評価制度の設計コンサルティングも併せて提供
- 弱み: AIの活用範囲は「目標設定文章の改善提案」「評価コメントのドラフト生成」にとどまり、業務データからの自動評価ではない
- 参考になる点: 日本市場でのAI×人事評価の温度感。「AI支援」までは受容されるが、「AI自動評価」はまだ市場に出ていない

### 主要トレンド

1. **AIは「自動評価」ではなく「評価支援」が主流**: Latticeの炎上事例が示す通り、「AIが人を評価する」ことへの社会的抵抗は強い。現時点で市場に受容されているのは「AIがドラフトを作成し、人間が判断する」モデル
2. **Rippling Talent Signalの登場が転換点**: 業務データから直接AIがパフォーマンスシグナルを生成するモデルが2024年に初めて市場に出た。ただし「入社90日の新人」に限定しており、全社員の継続評価には慎重
3. **EU AI Act のハイリスクAI規制**: 雇用・人事領域のAIは「ハイリスク」に分類され、透明性・説明可能性・バイアス監査が義務化。日本でも同様の規制導入が議論されており、参入障壁と参入機会の両面がある
4. **日本市場はまだ「デジタル化」段階**: カオナビ・HRBrainが解決しているのは「紙・Excelの評価プロセスをクラウド化する」こと。「AIが評価する」は次のフェーズだが、日本企業の受容度は未知数
5. **バイアス検知が差別化要因に**: Effy AIのようにバイアス検知を前面に出すアプローチが、AI人事評価への信頼性構築において重要になりつつある

## 技術的な実現手段

### LLM / AI基盤
- **OpenAI GPT-4o / GPT-4.5**: テキスト分析・評価文章生成に利用可能。API料金: GPT-4oで$2.50-$10/1M tokens
- **Anthropic Claude Sonnet 4.6 / Opus 4.6**: 長文コンテキスト分析に強み。Slack履歴や議事録の大量テキスト処理に適する
- **Azure OpenAI Service**: エンタープライズ向けにデータの地理的制約やコンプライアンス要件を満たす場合に有効

### データ連携API
- **Slack API**: conversations.history でチャネルの発言履歴を取得可能。レート制限あり（Tier 3: 50+リクエスト/分）。Enterprise Gridプランでは管理者APIで全チャネルアクセス可能
- **Microsoft Teams Graph API**: チャットメッセージ・会議出席データの取得が可能。管理者同意が必要
- **Google Workspace API**: Gmail・Drive・Calendar・Meetのデータ取得。ドメイン全体の委任設定が必要
- **Salesforce API**: 商談・受注データの取得。REST APIで各種オブジェクトにアクセス可能
- **HubSpot API**: CRMデータの取得。無料プランでもAPI利用可能

### 会議録文字起こし
- **Otter.ai API**: 会議の自動文字起こし+話者分離。API提供あり
- **tl;dv / Fireflies.ai**: Zoom・Google Meet・Teams会議の自動録音・文字起こし
- **OpenAI Whisper**: 自前で文字起こしパイプラインを構築する場合のOSS選択肢

### NLP・テキスト分析
- **LangChain / LlamaIndex**: LLMアプリケーション構築フレームワーク。RAGパイプラインで大量テキストの構造化分析が可能
- **spaCy / GiNZA**: 日本語テキストの前処理・固有表現抽出

### インフラ・セキュリティ
- **SOC 2 Type II / ISO 27001**: 人事データを扱うSaaSとして必須の認証
- **AWS / GCP**: VPC内でのデータ処理、暗号化、アクセス制御
- **プライベートLLMデプロイ**: Azure OpenAI Service や AWS Bedrock でデータの外部送信を最小化
