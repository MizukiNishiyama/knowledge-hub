# サマリー

updated_at: 2026-03-05

## アイディア概要（1-2文）

Slack・CRM・会議録等の業務データをLLMで統合分析し、人事評価を自動化するAIベースOS。既存の人事評価における属人性・ばらつき・工数の問題をAIで解決する構想だが、Rippling Talent Signal（2024年リリース）が先行しており、社会的受容・法的リスク・プラットフォーマー追随リスクの観点から、現時点でのNo-Go判断。

## 現状の評価

- 魅力度: ★★★☆☆（ピープルアナリティクス市場は年12-15%成長。ペインは実在するが、「AIが人を評価する」ことへの社会的受容が追いついていない）
- 実現可能性: ★★☆☆☆（技術的にはLLM API + Slack/CRM連携で構築可能だが、法的リスク・コンプライアンスコスト・信頼性構築のハードルが極めて高い。スタートアップの体力では対応困難）
- 差別化: ★☆☆☆☆（Rippling Talent Signalが$13.5B企業として先行。日本市場ではカオナビ・HRBrainがプラットフォームとしてAI機能を自社追加する方が合理的。独立プロダクトとしてのモートは構築困難）
