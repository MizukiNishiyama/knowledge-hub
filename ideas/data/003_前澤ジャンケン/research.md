## 前澤ジャンケンの詳細分析

### サービス概要
前澤友作氏が運営するarigatobankアプリ内のサービス。ユーザーは前澤氏とじゃんけんで対戦し、連勝するほど高額賞金を獲得できる。敗北時は動画広告を視聴して再チャレンジ可能（1日最大10回）。

### 賞金テーブル
| 連勝数 | 賞金 |
|--------|------|
| 1勝 | 1円 |
| 5勝 | 5円 |
| 10勝 | 100円 |
| 12勝 | 1万円 |
| 13勝 | 100万円 |
| 15勝 | 1,000万円 |

15連勝の確率: 1/32,768（約0.003%）。2023年4月時点で1,000万円獲得者は0人。

### ビジネスモデルの3本柱
1. **動画リワード広告収入**: 敗北→広告視聴→再チャレンジのループで広告インプレッションを量産。AnyManagerによるメディエーション最適化で運用コストを従来の1/12に削減
2. **提携カード代理店業**: 当選金受取にarigatobankカード（VISA）発行が必須。カード発行・利用に応じたインセンティブをカンム社から受領
3. **前澤氏のSNS影響力**: Twitterフォロワー1,000万人超を活用し、広告宣伝費ほぼゼロで集客

### 現在の状況
arigatobankの残高利用は**2025年9月30日で終了**。前澤じゃんけんも現在は休止状態。

### 批判・問題点
- 「確率が操作されている」疑惑（前澤氏は否定し法的対応を示唆）
- 広告視聴の搾取性への批判
- 前澤じゃんけんを装った詐欺サイトの横行
- 賞金受取にカード発行が必須な点への不満

## 競合サービス

### 1. モッピー（moppy）
- URL: https://pc.moppy.jp/
- 概要: 国内最大級のポイントサイト。広告案件経由でポイント獲得し、現金・電子マネーに交換
- ターゲット: ポイ活初心者〜上級者、副業志望者
- 価格帯: 無料。自社ASP「AD.TRACK」により高還元率を実現
- 強み: 累計会員1,300万人、2024年12月期売上高約125億円（セレス連結276億円）、上場企業の信頼性
- 弱み: エンタメ性が低い（案件クリック型が中心）
- 参考になる点: 自社ASPによる中間マージン排除モデル

### 2. ハピタス（Hapitas）
- URL: https://hapitas.jp/
- 概要: ネットショッピング経由のポイント還元に特化
- ターゲット: ネットショッピング利用者、主婦層
- 価格帯: 無料。EC事業者からの広告手数料を還元
- 強み: 会員500万人超、提携EC4,000社、流通総額1,790億円超（2023年度）
- 弱み: ゲーム・くじ要素がほぼない
- 参考になる点: 「ショッピングテイメント」のコンセプト

### 3. ポイントインカム（Point Income）

- 強み: 会員500万人超、独自ランク制度
- 弱み: ゲーム・くじ要素は限定的
- 参考になる点: 2025年9月にセレスが約2億円で買収（業界再編の動き）

### 4. トリマ（Torima）

### 法規制

#### 景品表示法
- **オープン懸賞**（商品購入等を条件としない）: 2006年4月に上限額規制が**撤廃**。無料アプリのくじはこれに該当する可能性が高く、法的ハードルは低い
- **クローズド懸賞**: 取引価額5,000円未満は景品上限が取引価額の20倍、5,000円以上は上限10万円

#### 賭博法との関係
- ユーザーが参加に金銭を支払わない場合、「財産上の利益の得喪を争う」要件を満たさず、賭博罪に該当しない
- 広告視聴は「財産上の利益」とは見なされないのが一般的見解
- ただし有料課金要素を含むと法的リスクが増大

## 技術的な実現手段

### Webアプリで使える広告配信手段

| 手段 | 特徴 | 備考 |
|------|------|------|
| **Google Ad Placement API** | AdSense/AdMobをHTML5ゲーム・Webアプリに統合。`adBreak()`でリワード広告表示、`adViewed`コールバックで視聴完了検知 | 最も実用的な選択肢 |
| **Google AdSense H5 Games Ads** | HTML5ゲーム向けに特化したAdSenseフォーマット | PWA対応可能 |
| **WebView + ネイティブSDK** | WebView内でAdMob/AppLovin等のネイティブSDKを呼び出すハイブリッド | eCPMは高いがWebアプリ単体では不可 |
| **AnyManager等のメディエーション** | 複数広告ソースを統合・最適化 | ARIGATOBANKが実際に採用 |

### 主要広告SDK比較

| SDK | eCPM目安（リワード動画） | 特徴 |
|-----|--------------------------|------|
| Google AdMob | $8-20 | 最大級のネットワーク、Web対応あり |
| AppLovin (MAX) | $15-30 | AI駆動の最適化、メディエーション強力 |
| Unity Ads | $10-25 | ゲーム開発者向け |
| IronSource | $30-80 (Android) | オファーウォール＋リワード動画 |
| maio | CPM 1,000円以上 | 日本国内特化、国内広告主案件が豊富 |

### くじ・確率制御の実装
- **重み付き乱択アルゴリズム**: 各賞に重みを設定し、サーバーサイドで暗号学的擬似乱数生成器（CSPRNG）を使用
- **Walker's Alias法**: O(1)での抽選が可能な最適アルゴリズム
- **確率表の公開**: 透明性確保のため必須（前澤ジャンケンの「操作」批判を回避）
- **抽選ログの保存**: 監査可能性の確保

### 不正防止
- **デバイスフィンガープリンティング**: 画面解像度、ブラウザ、OS等でデバイスを識別（IPQualityScore, Stytch等）
- **SMS/電話番号認証**: 1ユーザー1アカウント制限
- **行動分析**: クリックパターン・操作速度の異常検知
- **レートリミティング + CAPTCHA**: Bot対策
- **換金時KYC**: 本人確認で不正アカウント排除
- **専門ツール**: HUMAN Security, TrafficGuard, CHEQ, Anura
