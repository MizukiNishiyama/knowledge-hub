## 競合サービス

### 1. BATONZ（バトンズ）
- URL: https://batonz.jp/
- 概要: 国内最大級のM&A・事業承継マッチングプラットフォーム。売り手・買い手をオンラインでマッチング
- ターゲット: 事業承継を検討する中小企業経営者（売り手）と買収候補企業・個人（買い手）
- 価格帯: 売り手は成約まで無料。買い手はプレミアム会員5,390円/月（個人）、成約時に成功報酬（取引額の数%）
- 強み: 累計成約件数国内最大級、金融機関・士業との連携網、IT導入補助金対象
- 弱み: マッチング後の経営支援（ハンズオン）はカバーしていない。純粋なマッチングプラットフォーム
- 参考になる点: 売り手無料モデルで売り手側の参入障壁を下げる設計

### 2. M&Aキャピタルパートナーズ
- URL: https://www.ma-cp.com/
- 概要: 着手金ゼロ・成功報酬型の大型M&A仲介会社。戦略系コンサル出身者が多い
- ターゲット: 売上3億円以上の中堅・中小企業のM&A
- 価格帯: 着手金無料、成功報酬（レーマン方式、取引額の数%〜数十%）
- 強み: 高い成約率、大型案件の実績、コンサル的なアドバイザリー機能
- 弱み: 中小案件には対応しにくい、売却前の経営改善支援は提供していない
- 参考になる点: 着手金ゼロで信頼を獲得するモデル

### 3. ストライク（SMART）
- URL: https://www.strike.co.jp/
- 概要: M&A仲介とオンラインマッチングプラットフォーム「SMART」を運営。年間275件（2025年）の成約実績
- ターゲット: 中堅・中小企業のM&A。特に後継者問題を抱える企業
- 価格帯: SMART上の案件掲載・問い合わせは無料、仲介手数料は成功報酬型
- 強み: 全国の金融機関・公認会計士・税理士との連携ネットワーク、2035年まで市場拡大が見込まれる事業承継案件に注力
- 弱み: M&A成立前の経営支援・企業価値向上支援は薄い
- 参考になる点: 専門家ネットワークを活用した信頼性の高いエコシステム

### 4. サーチファンド・ジャパン（JaSFA）
- URL: https://japan-sfa.com/
- 概要: 日本初のサーチファンドモデル支援機関。サーチャー（次世代経営者候補）が投資家から資金を集め、買収先企業を探してCEOになるモデル
- ターゲット: 30代前後で起業・経営を志す人材（サーチャー）と、次世代経営者育成に関心のある投資家
- 価格帯: ジャパン・サーチファンド・プラットフォームへのLP出資（コミット総額58億円超）
- 強み: 日本最大のサーチファンド支援組織、DBJ（日本政策投資銀行）・中小機構などの公的機関が参画、一気通貫サポート
- 弱み: サーチャー育成が中心でありM&A売却前の現経営者支援は主目的ではない、スキームが複雑で時間がかかる
- 参考になる点: 「先輩（フェロー）」概念に最も近い構造。投資家がメンターとしてサーチャーを支援

### 5. 事業承継ファンド（東京都産業労働局ほか）
- URL: https://www.sangyo-rodo.metro.tokyo.lg.jp/chushou/kinyu/fund/
- 概要: 公的機関が組成・出資する事業承継特化型ファンド。後継者不在企業の株式を買い取り、企業価値向上後に売却する
- ターゲット: 後継者問題を抱える中小企業（主に売上5億〜30億円規模）
- 価格帯: ファンドがエクイティ投資（株式取得）。投資回収は3〜5年後のM&A売却時
- 強み: 公的資金の裏付け、ハンズオン支援（財務・人事・販路拡大）が充実
- 弱み: 手続きが複雑・時間がかかる、ファンド側が経営権を握るため現経営者の主導権が失われる
- 参考になる点: 「支援→企業価値向上→M&A売却→リターン分配」のスキーム構造が最も類似

### 6. M&A総合研究所
- URL: https://masouken.com/
- 概要: AIを活用したM&Aマッチング・仲介サービス。最短3ヶ月での成約を標榜
- ターゲット: 中小企業の事業承継・成長M&A
- 価格帯: 着手金無料・成功報酬型、料金はレーマン方式（取引額の数%）
- 強み: AIによるマッチング効率化、スピード成約、透明性の高い料金体系
- 弱み: 企業価値向上のためのハンズオン支援はなし
- 参考になる点: AI活用による仲介コスト削減モデル

## 市場規模・トレンド

### TAM / SAM / SOM 推定

| 区分 | 市場規模 | 根拠 |
|---|---|---|
| **TAM（日本の事業承継・M&A潜在市場）** | 約13兆5,000億円 | 事業承継M&A潜在需要（日経新聞, 2024年1月） |
| **SAM（M&Aを検討中の中小企業経営者向け支援市場）** | 約1兆〜2兆円 | 2025年に70歳超の経営者約245万人のうち約127万人が後継者未定。うち一定割合がM&Aを選択 |
| **SOM（フェロー型M&A前提支援の初期獲得可能市場）** | 約100億〜500億円 | サーチファンド・事業承継ファンド・M&A仲介の隣接市場のうち、ハンズオン支援特化の部分。JaSFAのコミット総額58億円超が参考値 |

### 市場トレンド

1. **M&Aエグジットの主流化**: 2025年上半期のM&A市場規模は約31兆円（前年同期比3.6倍）。スタートアップのIPO以外のイグジットとしてM&Aが急速に普及
2. **後継者問題の深刻化**: 2025年までに70歳超の中小企業経営者約245万人、うち半数の127万人が後継者未定。廃業リスクのある黒字企業が増加
3. **ETA（Entrepreneurship Through Acquisition）の台頭**: 2023年に過去最高の94件のサーチファンドが世界で設立。日本でもJaSFAなど先行プレイヤーが成長
4. **base44の日本市場参入（2026年3月）**: Wix傘下のbase44が2026年3月に日本語版を正式展開。AIノーコードアプリ開発のハードルが急低下し、このようなMVPが急増中
5. **ロールアップ戦略の普及**: 分散した中小企業を連続買収して統合・企業価値向上するPE型のロールアップ戦略が調剤薬局・物流・IT等の業界で活発化

## 「M&A前提の支援スキーム」の構造分析

### 類似する既存スキームの比較

| スキーム | 主役 | 支援内容 | リターン | リスク |
|---|---|---|---|---|
| **エンジェル投資** | 個人投資家 | 資金・アドバイス | IPO/M&A時の株式売却益 | 投資先倒産リスク |
| **VC投資** | ファンド | 資金・ネットワーク・ハンズオン | IPO/M&A時のキャピタルゲイン | 投資先倒産リスク、ドラッグアロング |
| **サーチファンド** | サーチャー（次世代経営者） | 買収資金・経営移行サポート | 5〜7年後のM&A/IPO | 買収先の経営悪化リスク |
| **事業承継ファンド** | PEファンド | 株式取得・ハンズオン経営支援 | 3〜5年後のM&A売却益 | 企業価値向上失敗リスク |
| **Senjin Fellowship（推測）** | 「先輩」経営経験者 | アドバイス・ネットワーク・資金？ | M&A成立時の成功報酬または株式 | M&A不成立リスク |

### 法的・制度的論点

1. **金融商品取引法（集団投資スキーム）**: フェローが複数の経営者に出資し、その運用利益を分配する場合、第2条2項5号の「集団投資スキーム」に該当する可能性がある。ファンド規制の対象となれば第二種金融商品取引業の登録が必要
2. **出資法**: 「M&A売却時のリターンを前提に出資を受ける」スキームは、預かり金・出資金の形態によっては出資法（出資の受入れ、預り金及び金利等の取締りに関する法律）の規制対象になる可能性
3. **M&A仲介業規制**: M&Aの媒介・代理を業として行う場合は、M&Aシニアエキスパートや仲介業者登録が求められる場合あり（現状は任意登録制だが規制強化の議論がある）
4. **利益相反**: フェロー（支援者）がM&A成立を急ぐことで、売り手経営者に不利な条件での売却を促すリスクがある

## 技術的な実現手段

- **現状**: base44（AIノーコードプラットフォーム）でMVP構築済み
- **スケール時の技術スタック候補**:
  - バックエンド: Supabase（PostgreSQL + 認証 + リアルタイム）、Firebase
  - フロントエンド: Next.js + React
  - マッチングロジック: OpenAI API（プロファイル分析・相性スコアリング）
  - 契約管理: DocuSign / Cloudsign API連携
  - CRM: HubSpot / Salesforce（フェローと経営者の関係管理）
- **base44の継続利用**: 月数万円〜で本番運用が可能。ただしWix買収後の料金改定リスクあり
