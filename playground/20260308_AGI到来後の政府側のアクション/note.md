# Role
国家戦略アドバイザー、および先端技術・経済政策アナリスト

# Objective
AGI（汎用人工知能）の台頭が日本国家に及ぼす多角的な影響を特定し、国が主導すべき具体的なアクションプランを策定する。特に、労働市場の変化に伴う社会保障制度の再構築と、その財源確保に関する最新の議論（論文・学術的知見）を整理する。

# Scope
* 地理的範囲：日本国内（およびグローバルな経済・技術競争環境）
* 時間的範囲：現在から2030年代後半までの移行期
* 対象分野：労働経済、社会保障制度、国家財政、技術ガバナンス、産業構造

# Key Questions
1. AGIの社会実装により、日本固有の雇用慣行や産業構造にはどのような破壊的変化が生じるか？
2. 大規模な失業・スキルの陳腐化が発生した場合、既存のセーフティネットで対応可能か？
3. AGI時代における「ベーシックインカム（BI）」や「負の所得税」の有効性と、その導入障壁は何か？
4. BI等の社会保障を支えるための新たな財源（ロボット税、データ課税、資本利得増税等）に関する最新の国際的・学術的な議論はどのようになっているか？
5. 日本政府が国際競争力を維持しつつ、国民の生活を保障するために優先すべき「国家主導アクション」は何か？

# Research Tasks
STEP1：AGIの定義と、日本における導入シナリオ（タイムライン）の整理
STEP2：日本経済への影響分析（失業率推移予測、生産性向上のポテンシャル、格差拡大の構造）
STEP3：社会保障リデザインの検討（ベーシックインカム、リスキリング支援、公共サービスの再定義）
STEP4：財源確保策の深掘り（最新論文・提言に基づく税制改革、通貨発行益、デジタル資産課税等のリサーチ）
STEP5：地政学的・競争戦略的視点からの国家アクション（国産AI開発、規制枠組み、国際標準化）の列挙
STEP6：構造化（ロードマップおよび優先順位付けされた政策パッケージの作成）

# Analytical Framework
* **経済構造：** 労働分配率の低下と資本分配率の上昇
* **制度・財政：** 既存の社会保険方式から税方式・給付方式への転換
* **技術・ガバナンス：** AGIの安全性、倫理、および計算資源の国家保有
* **社会心理：** 労働の尊厳、余暇の活用、および社会のレジリエンス

# Output Format
以下の構成でMarkdown形式で出力すること。
1. **概念整理：** AGIが日本国家に与える「正・負」の影響サマリー
2. **分類体系：** 想定される課題（経済、社会、政治、技術）の整理
3. **比較表：** 提唱されている主な財源確保案（税目、メリット、デメリット、最新論文での評価）
4. **構造ツリー：** 日本政府がとるべきアクションの体系図
5. **提言：** 2026年時点での最新知見に基づく具体的ロードマップ

# Constraints
* 最新の経済学論文、国際機関（OECD, IMF等）のレポート、政府の有識者会議資料をベースにすること。
* 財源論については、単なるアイデアではなく、実現可能性と経済的影響（インフレ懸念等）の議論を反映させること。
* 日本特有の人口減少・少子高齢化という文脈を必ず加味すること。
* 不明な事実を捏造せず、論点として提示すること。
