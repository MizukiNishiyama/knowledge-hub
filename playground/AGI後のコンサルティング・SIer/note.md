# Role
戦略コンサルタント、およびIT産業アナリスト（テクノロジー経済学専門）

# Objective
AGI（汎用人工知能）の普及と開発コストの劇的な低下（DeepSeek R1等の事例）が、日本のコンサルティングおよびSI（システムインテグレーション）市場に与える長期的影響を分析する。特に「内製化の進展による市場縮小」と「ジェボンズのパラドックスによる需要拡大」の相反する力が、市場規模と構造をどう変容させるかを予測する。

# Scope
* 調査対象：日本のITコンサルティング市場、SI市場、ソフトウェア開発産業
* 比較対象：過去の自動化（RPA、ノーコード、クラウド移行）による市場変化
* 視点：コスト効率化 vs 新規需要創出（ジェボンズのパラドックス）

# Key Questions
1. ソフトウェア開発の「単位あたりのコスト」が激減した際、企業のIT投資予算は削減されるのか、それともより高度な領域へ再配分されるのか？
2. 「内製化」が容易になることで、外部ベンダー（SIer）に求められる役割はどう変化するか？（単なる実装から、アーキテクチャ設計やAI統治へのシフト等）
3. AGIによる効率化が「飽き足りることのない需要（Jevons Paradox）」を生む具体的な領域（例：超個別化サービス、リアルタイム経営最適化）はどこか？
4. DeepSeek等の「低コスト・高効率モデル」の台頭は、計算資源（チップ・電力）需要と、それを活用するサービス市場の相関関係をどう変えるか？
5. 2030年時点でのSIer/コンサルティングファームの「生き残りモデル」とはどのような形態か？

# Research Tasks
STEP1：現状のコンサル・SI市場の収益構造（人月単価・工数依存型）の整理
STEP2：AGIによる「開発・分析の自律化」がもたらす代替可能性の検証（どの工程が消滅するか）
STEP3：「ジェボンズのパラドックス」の歴史的検証とAI市場への適用（蒸気機関、インターネット通信、クラウドとの比較）
STEP4：内製化トレンドの限界（人材確保、法的責任、技術更新スピード）の特定
STEP5：市場規模のシナリオ予測（単純縮小シナリオ vs 需要爆発による再成長シナリオ）
STEP6：構造化（SIer・コンサルの「提供価値の変遷図」の作成）

# Analytical Framework
* **ジェボンズのパラドックス（Jevons Paradox）：** 効率向上による需要の反転増
* **アンバンドリングとリバンドリング：** 既存サービスの解体と再構築
* **取引コスト理論：** 内製（Make）か購入（Buy）かの境界線の変化
* **レイヤー構造の変化：** インフラ（チップ）、プラットフォーム（AGI）、アプリケーション（SI）の利益配分

# Output Format
1. **市場変容の概念整理：** 効率化が市場を「縮小」させる要因と「拡大」させる要因の対置
2. **分類体系：** AGI時代に「消滅するサービス」と「新設されるサービス」の一覧
3. **比較表：** 従来型SIer vs AGIネイティブ型SIerのビジネスモデル比較
4. **構造ツリー：** ジェボンズのパラドックスがIT市場全体を押し上げるメカニズムの図解
5. **結論：** コンサル・SI市場の「質的変化」と2030年の市場予測

# Constraints
* ジェンスン・ファン氏、サティア・ナデラ氏の最新の発言およびDeepSeekショックの影響を前提に置くこと。
* 単なる楽観論・悲観論に終始せず、経済学的原理（需要の価格弾力性など）に基づいた論理を展開すること。
* 日本特有の「IT人材不足」と「レガシーシステム」が変数としてどう作用するかを含めること。
