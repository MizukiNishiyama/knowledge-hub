# 丸川珠代

| 項目 | 内容 |
|------|------|
| 氏名 | 丸川珠代（まるかわ たまよ） |
| 生年月日 | 1971年1月19日 |
| 年齢 | 55歳 |
| 院 | 衆議院 |
| 選挙区 | 東京都第7区（港区・渋谷区） |
| 当選回数 | 1回（衆議院）、参議院3期 |

## 2. 経歴（時系列）
- 1971: 兵庫県生まれ
- 1993: 東京大学経済学部卒業、テレビ朝日入社（アナウンサー）
- 2007: 第21回参議院議員通常選挙で初当選
- 2013: 参議院2期目当選
- 2015: 環境大臣就任（COP21パリ協定合意に関与）
- 2016: 東京オリンピック・パラリンピック担当大臣
- 2019: 参議院3期目当選
- 2021: 内閣府特命担当大臣（男女共同参画）兼五輪担当大臣（菅内閣）
- 2024: 参議院議員を自動失職し衆院東京7区に出馬（落選）
- 2026: 第51回衆議院議員総選挙で東京7区から初当選

## 3. 歴任ポスト
| 期間 | ポスト |
|------|--------|

## 4. 政策スタンス・専門領域
- 環境政策（COP21パリ協定）
- 男女共同参画
- オリンピック・パラリンピック推進
- メディア・広報

## 5. 主な功績・実績
- COP21でパリ協定合意に貢献（環境大臣として）
- 東京五輪の準備・運営を担当大臣として推進
- テレビアナウンサーから政治家への転身

## 6. 第三者評価・注目点
- 元テレビ朝日アナウンサーという高い知名度
- 参議院から衆議院へのくら替え
- 旧安倍派の裏金問題（2018-2022年に計822万円のキックバック）が報じられた
