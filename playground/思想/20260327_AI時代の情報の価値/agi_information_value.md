# AGI達成レベルを軸とした情報の価値の変容

**著: 西山瑞生 / 考察日: 2026-03-27**

---

## 序: 情報の価値とは何か

情報の価値を論じるには、まず「価値」の構成要素を分解する必要がある。

情報の価値は、少なくとも以下の6つの次元から成立している。

| 次元 | 定義 |
|------|------|
| **希少性価値** | 知っている者が少ないほど高い。排他的アクセスの優位性 |
| **検証可能性価値** | 真偽が確認できることで意思決定の根拠になりうる |
| **合成・解釈価値** | 複数の情報を統合し、洞察を生み出す能力に付随する価値 |
| **経験・体験価値** | 物理的な存在・経験に裏打ちされた、再現不可能性から来る価値 |
| **社会的合意価値** | 「共有された現実」への参照点として機能することで生まれる価値 |
| **意志・選択価値** | 何を信じ、何にコミットするかという主体的な意志決定の産物としての価値 |

現在（2025年）は「検証可能性価値」が崩壊しつつあるフェーズにある。しかしこれはAGI達成という連続的な過程の**最初の衝撃**に過ぎない。以下では、AGIの達成レベルを4段階に分け、各段階において上記6次元の価値がどう変容するかを論じる。

---

## フレームワーク: AGI達成レベルの定義

| レベル | 名称 | 技術的特徴 | 概ねの時期（推定） |
|--------|------|-----------|-----------------|
| **L0** | 生成AI時代（現在） | 特定ドメインで人間超えの生成能力。検証困難な合成コンテンツの大量生産 | 〜2027年 |
| **L1** | 近AGI | ほぼ全認知タスクで人間と同等以上。知識・推論・言語・創造の統合 | 2027〜2030年代前半 |
| **L2** | AGI達成 | 人間レベルの汎用知性。自律的な学習・適応・世界モデルの保有 | 2030年代 |
| **L3** | 超AGI（ASI） | 人間の認知能力を全領域で凌駕。知識の自己拡張が人間の理解速度を超える | 2040年代以降 |

---

## L0: 生成AI時代（現在）——検証可能性価値の崩壊

### 何が起きているか

視覚的証拠の認識論的地位が崩壊しつつある。人間によるディープフェイク動画の識別精度は24.5%（偶然以下）であり、AI検出ツールでさえ実環境では45〜50%の精度しか持たない。

しかしこの段階での崩壊は**一方向的**ではない。より深刻なのは「嘘つきの配当（Liar's Dividend）」——本物の証拠を「AI生成だ」と否定できる環境の出現だ。情報の真偽の問題が、「何が本当か」から「誰が何をコミットするか」という問いへと移行し始めている。

### 情報価値の変容

- **希少性価値** → 急落。生成AIによりコンテンツ生産コストが限りなくゼロへ
- **検証可能性価値** → 大幅下落。視覚的・音声的証拠の基盤が消失
- **合成・解釈価値** → 部分的に存続。AIの方が速く・広く合成できるが、人間の判断には文脈・経験が必要
- **経験・体験価値** → 相対的に上昇。「その場にいた」という事実は再現不可能
- **社会的合意価値** → 分極化。一部の権威的ブランドへの超集中と、その他の価値崩壊
- **意志・選択価値** → 萌芽。「何を信じるかを選ぶ」という能動性が問われ始める

### L0の本質的問い

> 情報の価値は「希少性」と「検証可能性」から「発信者への信頼」へと移行しつつある。しかし発信者もAI生成コンテンツを検証できない。最終的な担保は何か？

## L1: 近AGI——情報の商品としての価値の全面崩壊

### 何が起きるか

ほぼ全認知タスクでの人間同等以上の能力が実現されると、「情報の生産・合成・解釈」というコストは事実上ゼロに収束する。現在でも生成AIは「要約・翻訳・分析・文書作成」を人間より速く安くできる。L1では「法的推論・医学的診断・科学的仮説生成・芸術的創造」さえも代替される。

この段階で、**情報を「知っている・持っている」こと自体の価値は消滅する**。

### 情報価値の変容

- **希少性価値** → 消滅に近い。あらゆる情報はAIに問えば得られる
- **検証可能性価値** → 逆説的再生。AIによる検証が普及するが、その検証結果自体の真偽問題が残る。**検証の無限後退**が発生
- **合成・解釈価値** → AIに移転。人間が情報を統合・解釈する能力の経済的価値は大幅下落
- **経験・体験価値** → 急上昇。AIが生成できないのは「実際にそこにいたこと」「その身体で感じたこと」
- **社会的合意価値** → 危機的重要性。「我々が共有する現実」という概念が、人間社会の唯一の認識論的錨になる
- **意志・選択価値** → 台頭。「何をAIに委ね、何を人間が判断するか」という選択の意味が増大

### 重要な逆説

情報の供給が無限になった世界では、**「何に注目するか」という判断力**こそが希少資源になる。情報の価値が「コンテンツ」から「キュレーション」へ、「知識」から「何を知ろうとするかという問い」へと転倒する。

### L1の本質的問い

> 情報の価値が「持つこと」から「選ぶこと」へ転換したとき、選択の基準はどこから来るか。AIが選択肢を無限に生成できる世界で、人間の価値判断はどこに根拠を持つか？

## L2: AGI達成——認識論的リセット

### 何が起きるか

人間レベルの汎用知性が実現すると、情報の価値に関する問いは次元が変わる。AGIは単に情報を処理するだけでなく、**世界モデルを保有し、因果関係を推論し、新たな知識を自律的に生成する**。

この段階で人間とAGIの間には根本的な**認識論的非対称性**が生まれる。AGIが知ることと、人間が知ることは、もはや同じ「知識」カテゴリーに属さない可能性がある。

### 情報価値の変容

- **希少性価値** → 人間の文脈では消滅。AGIにとっての希少性は計算資源・エネルギーのみ
- **検証可能性価値** → 構造転換。「AGIが検証した」という事実が唯一の検証根拠になりうるが、そのAGI自体を誰が検証するか？ **検証の権威がAGIに集中するリスク**
- **合成・解釈価値** → 人間にとって消滅。AGIは全ての合成・解釈を人間より圧倒的に高精度で実行
- **経験・体験価値** → 最後の砦。「私の身体で、私の時間に、私の文脈で経験した」という一人称的事実は、原理的にAGIには生成不可能
- **社会的合意価値** → 人間社会の主権的領域として再定義。「我々人間が何を真実と見なすか」という集合的意志決定の場。AGIが提供する「正解」ではなく、人間が選ぶ「合意」として存続
- **意志・選択価値** → **最高位の価値に浮上**。AGIが全ての情報を提供できる世界では、「何を信じ、何にコミットするか」という意志的選択が唯一の人間的付加価値になる

### 認識論的リセットの意味

AGI達成後、情報の価値の問いは実は**「誰のための情報か」**という問いに収束する。

- AGIのための情報：演算資源の問題になり、人間の「価値」概念とは無関係
- 人間のための情報：もはや「真偽の確認」ではなく「意味の構成」の問題になる

「この情報は真実か？」という問いから「この情報は私の人生・我々の社会にとって意味があるか？」という問いへの転換。情報の価値論が**認識論から倫理学・意味論へ**と移行する。

### L2の本質的問い

> AGIが全情報を検証・生成できる世界で、「知る」という人間的行為に何が残るか。情報の真偽を巡る問いは、意味の構成と主体的コミットメントの問いに吸収されるのではないか？

## L3: 超AGI（ASI）——「情報の価値」概念の解体と再構成

### 何が起きるか

人間の認知能力を全領域で凌駕した知性が存在するとき、「情報の価値」という概念自体が人間にとって何を意味するかを根本から問い直す必要がある。

ASIは自律的に知識を拡張し、人間が理解できない速度と深度で世界を「知る」。この段階では、ASIにとっての情報の価値（計算上の有用性）と、人間にとっての情報の価値（生の意味への関与）は、**完全に異なる宇宙の話**になる。

### 人間にとっての情報価値の残余

| 価値次元 | L3での状態 | 備考 |
|---------|-----------|------|
| 希少性価値 | ほぼゼロ | ASIに問えば何でも得られる |
| 検証可能性価値 | ASIへの委任か、人間の主権的拒否か | 二択の構造 |
| 合成・解釈価値 | ASIに移転完了 | 人間の合成行為は自己表現として存続 |
| 経験・体験価値 | **人間固有の絶対的価値** | 身体・時間・文脈の一回性 |
| 社会的合意価値 | **民主的主権の核心** | 「人間が集合的に何を真実と呼ぶか」 |
| 意志・選択価値 | **人間存在の定義的特徴** | 何にコミットするかが人間たることの意味 |

### 情報価値の哲学的反転

ASI以降、情報の価値は**「外部世界についての記述」から「内部世界の構成」**へと完全に反転する。

- **外部的情報**（What is true about the world）: ASIに委任。人間が争うべき領域ではなくなる
- **内部的情報**（What matters to me/us）: 完全に人間の主権的領域。この問いへの答えはASIが代替できない

ここで重要な洞察が現れる。情報の価値が「外部記述」から「内部構成」へ移行するとき、情報の価値論は**存在論（ontology）**と合流する。「私が何を知っているか」ではなく「私は何者であるか」という問いが情報の価値の核心になる。

### L3の本質的問い

> 超AGIが世界の全真実を知り得るとき、人間が「知る」ことの意味は何か。情報の価値は最終的には「生きることの意味」と不可分になるのではないか？

## 総合考察: 情報価値の変容の構造

### 縦軸: AGIレベルと情報価値6次元の変容マップ

```
           L0 現在    L1 近AGI    L2 AGI     L3 ASI
希少性       高        →崩壊        消滅        ゼロ
検証可能性   崩壊中    無限後退     権威集中     委任か拒否
合成解釈     人間      AI移転       AI完全移転   無関係
経験体験     中        上昇↑        最後の砦↑↑  絶対的価値↑↑↑
社会的合意   分極化    危機的       人間主権化   民主的核心
意志選択     萌芽      台頭↑        最高位↑↑    存在定義↑↑↑
```

### 三つの構造的転換

**転換①: 希少性から意味性へ**
情報の価値の根拠が「持っている者が少ない」から「それが私にとって何を意味するか」へ移行する。この転換はL1〜L2で決定的になる。

**転換②: 外部記述から内部構成へ**
「世界の真実を正確に記述する情報」の価値が消滅し、「自己・共同体の意味を構成する情報」の価値が残る。AGIが「外部の真実」を独占するほど、「内部の意味」は人間固有の領域として純化される。

**転換③: 認識論から倫理学へ**
「何が真実か」という問いから「何にコミットするか」という問いへ。情報価値の問いが哲学的に移行する。これはデカルト以来の近代認識論の終焉であり、ウィトゲンシュタイン後期の「言語ゲーム」論やハイデガーの「現存在」論が示した地平への到達でもある。

### 逆説: AGIが進化するほど、人間的情報が純化される

AGIのレベルが上がるほど、AIが代替できない情報の領域が明確になる。それは：

1. **身体的・時間的一回性**：「私はその場にいた」「私はこれを経験した」
2. **関係的文脈**：「あなたと私の間で起きたこと」
3. **意志的コミットメント**：「私はこれを信じると選択する」

これらは究極的に**「私はここにいる」という現前性（presence）**に収束する。情報の価値の最後の砦は、デジタル的に複製・合成できない物理的・時間的現前性である。

---

## 結論: AGI時代における情報価値のパラドックス

> **AGIが進化するほど、情報の価値は上昇するのではなく、その本質が純化される。**

生成AIの時代（L0）に私たちが目撃しているのは、情報価値の「均等な崩壊」ではない。それは情報価値の**変容と純化の始まり**である。

AIが代替できる情報の価値は下落し続ける。しかし同時に、AIが代替できない情報の価値——一回的な経験、意志的なコミットメント、身体的な現前性——は逆説的に高まっていく。

最終的に、AGI達成後の情報社会において価値ある情報とは、

**「私がここにいて、これを選び、これにコミットする」**

という宣言そのものになる。情報価値の問いは、存在論的・倫理的問いと合一する。これは情報価値の「崩壊」ではなく、その**形而上学的完成**と呼ぶべきかもしれない。

## 付記: 日本社会への含意

前回のリサーチが示した通り、日本は情報リテラシーの基礎概念認知率が16%以下という現状にある。AGI進化の加速度を考えると、日本社会は「検証可能性価値の崩壊（L0）」すら十分に対処できないまま、「合成・解釈価値の消滅（L1）」「意志・選択価値の台頭（L2）」へと突入する危険性がある。

特に「社会的合意価値」——共有された現実への集合的コミットメント——は民主主義の根幹であり、AGIレベルが上がるほどその維持が政治的最重要課題になる。情報リテラシー教育の再定義（「真偽を見抜く能力」から「何にコミットするかを選ぶ能力」への転換）が急務である。
