# people

人物リサーチを保存するディレクトリ。

## ディレクトリ構成ルール
- 人物ごとに `people/000_{人名}` の形式でディレクトリを作成する。
- 各ディレクトリ配下に、調査内容をまとめた `research.md` を置く。
- 番号は3桁連番（`000`, `001`, `002` ...）で管理する。

例:
- `people/000_人名/research.md`
- `people/001_渋沢栄一/research.md`

## リサーチ項目（評価軸）
- 名前
- ラベル（実業家、政治、宗教 など）
- 概要
- 経歴
- 行動原理
- 参考になる点
- 見習うべきでない点

詳細な運用方針はルートの `reference.MD` を参照。
